# 革命、反革命与复辟

> 文档性质：群众动员、双重权力、政权断裂、新秩序整合与长期反扑机制  
> 目标：革命是困难、可失败、可变形且需要治理能力的历史过程；反革命与复辟同样由物质利益和组织支撑。

## 1. 改革与革命的边界

改革是在现有主权和国家基本连续的条件下改变规则；革命是主权来源、阶级控制或国家机器发生断裂，替代组织开始独立征收、裁判、动员和使用暴力。

```text
政策激进 ≠ 革命
皇帝退位 ≠ 社会革命
改朝换代 ≠ 生产关系改变
群众暴乱 ≠ 革命胜利
革命夺权 ≠ 新制度稳定
```

当旧国家仍能独占税收、军队和任命时，即使进行均田、工业化或议政，也属于激进改革。地方组织能与旧国家并行提供粮食、治安、司法和军队时，才进入双重权力。

## 2. 动员阶梯

集体行动按组织和风险逐级解锁，不从不满直接跳到全国内战。

| 级别 | 行动 | 必要条件 | 国家应对 | 失败后果 |
|---:|---|---|---|---|
| M0 | 私下抱怨、逃役、隐匿 | 不满 | 忽视、个案惩处 | 结构压力继续积累 |
| M1 | 联名、上书、乡约议论 | 基础传播、合法渠道 | 受理、拖延、压制领袖 | 合法性升降、组织经验 |
| M2 | 抗税、抗租、罢市、逃亡 | 地方组织、共同利益 | 让步、分化、拘捕、调兵 | 地方经济和税收受损 |
| M3 | 跨县联盟、行业停工、军中请愿 | 中等组织、传播和领袖 | 谈判、紧急法、镇压 | 形成全国议题或地下化 |
| M4 | 地方接管、民团/兵变、替代裁判 | 国家失灵、武装或军队中立 | 军事清剿、自治妥协 | 双重权力萌芽 |
| M5 | 区域革命、联合政府、纲领战争 | 替代机构、资源和联盟 | 全面内战或政治重组 | 政权分裂、外部干预 |
| M6 | 全国夺权与制度重建 | 控制首都/财政/军队且有治理方案 | 旧集团反攻、妥协或流亡 | 整合成功、热月化、复辟或碎片化 |

每次行动留下“集体行动经验”。成功让步可降低不满但提高组织认可；镇压可降低公开组织却提高共同记忆和秘密网络；惨败会造成恐惧、领袖损失和路线分裂。

## 3. 革命机会的六个门槛

### 3.1 国家危机

至少两项严重失灵：国库/债务、欠饷、粮运、合法性、继承、战争、行政执行。单一灾荒不会自动革命。

### 3.2 精英分裂

宫廷、官僚、士绅、军队或商人中至少一部分拒绝继续支持旧秩序，并提供资金、信息、合法性或军队中立。

### 3.3 群众组织

小农、佃户、工人、城市居民或军户拥有跨地域组织、共同解释和行动经验。人口数量不能替代组织。

### 3.4 替代纲领

联盟能回答：土地归属、税制、债务、军队、地方权利、工业、宗主关系和统治者产生方式。只有口号没有制度安排，会在夺权后迅速分裂。

### 3.5 替代治理能力

至少能在核心区域征收、配粮、治安、任命和维持交通。地方会馆、村社、改革官僚、军政委员会或城市议会可提供不同能力。

### 3.6 强制力量裂解

旧军队的一部分倒戈、中立、欠饷或被新军牵制。没有这一项，大规模政治运动更可能被镇压或转入长期地下。

建议进入全国革命危机的组合：

```text
state_crisis ≥ 65
elite_split ≥ 55
mass_mobilization ≥ 60
alternative_capacity ≥ 45
旧军队可靠性 ≤ 55 或 革命武装能力 ≥ 45
且至少一个政治联盟提出完整纲领
```

## 4. 意识形态如何形成

意识形态不是科技点，而由三部分组成：

```text
既有政治语言
+ 对现实矛盾的解释
+ 可组织的制度方案
```

### 4.1 七个纲领轴

| 轴 | 一端 | 另一端 |
|---|---|---|
| 主权 | 皇统/天命 | 人民、议会、军队或地方共同体 |
| 土地 | 地权绝对 | 限田、均田、公共/合作土地 |
| 资本 | 私人自由 | 国家、合作或劳动控制 |
| 劳动 | 身份/家长控制 | 契约、保护、参与或共同管理 |
| 地方 | 中央一统 | 自治、联邦或多中心契约 |
| 动员 | 常规税役 | 国民征兵、群众组织或军政征发 |
| 发展 | 稳定农业 | 商业扩张、工业富强或民生技术 |

政治联盟从这些轴组合纲领，而不是从固定现代党派列表选择。

### 4.2 概念可用性

较现代的人民主权、劳动代表、公司法权或合作工业需要满足传播条件：

- 印刷与识字扩大；
- 科举、书院、技术学校或翻译网络传播新概念；
- 商会、工场、军队和地方议政提供组织经验；
- 旧制度多次无法解释或解决危机；
- 有人物把概念转写成当时可理解的名分和制度。

穿越者可以提出概念，但无法直接让全国接受。思想必须由人物、文本、组织和实践扩散；领袖被杀后，只有被制度和组织保存的部分继续存在。

## 5. 六类革命/反革命集团

### 5.1 皇统改革派

主张以皇帝授权进行均税、工业、军队和民生改革，保留国号与皇统。

- 优势：合法性和官僚连续性；
- 弱点：依赖君主与宫廷，容易在继承时回撤；
- 可形成君主立宪式、官僚集权式或民本君主式结果。

### 5.2 官僚共和/议政派

主张规则、议政、预算和专业官僚高于个人恩宠。

- 基础：改革官僚、士人、部分商人和城市精英；
- 优势：治理和文本能力；
- 弱点：群众基础有限，易演化为精英寡头或官僚“热月”。

### 5.3 商业市民派

主张产权、合同、贸易、城市自治、纳税代表和工业投资。

- 基础：商人、工场主、会馆、城市专业者；
- 优势：财政、信用和城市组织；
- 弱点：城乡差异、劳动冲突、被指责以财富买政治。

### 5.4 农民均平派

主张减租、债务取消、限田/均田、赈济和基层共同体。

- 基础：小农、佃户、流民、基层士人；
- 优势：人口与地方动员；
- 弱点：区域分散、全国财政和工业纲领不足、小业主与无地者分裂。

### 5.5 劳动共同派

主张工资、劳动保护、工场参与、公共工业和社会保障。

- 基础：城市雇工、工匠、矿工、运输劳动者、激进技术人员；
- 优势：控制关键生产和运输节点；
- 弱点：初期人数和地域有限、粮食依赖、内部行业差异。

### 5.6 军事救国/秩序派

主张军队统一、紧急权力、军工和国家生存优先，可成为革命盟友或反革命核心。

- 基础：将领、新军、军户、难民和军需集团；
- 优势：武装、纪律和交通；
- 弱点：财政依赖、军头化、民生与政治代表受压。

## 6. 革命联盟的组成规则

任何集团单独都难以全国夺权。可行联盟至少连接三种能力：

```text
政治合法性/文本能力
+ 财政与生产供给
+ 群众和地域组织
+ 军事或治安能力
```

| 联盟示例 | 优势 | 核心冲突 |
|---|---|---|
| 皇统改革 + 官僚 + 新军 | 国家连续、执行快 | 皇权边界、军队监督、群众参与 |
| 官僚议政 + 商业市民 + 城市劳动 | 财政、制度与城市控制 | 财产权、工资、选举/代表范围 |
| 农民均平 + 劳动共同 + 激进官僚 | 再分配和群众基础 | 城乡资源、专业治理、工业所有制 |
| 商业市民 + 新军 + 地方士绅 | 地方资源和武装 | 寡头化、中央统一、农村地租 |
| 农民均平 + 地方军头 | 快速推翻旧地方秩序 | 军头吞并运动、土地承诺背叛 |

联盟在夺权前可共享敌人，夺权后会围绕土地、军队、税收、资本和政治代表分裂。玩家必须提前写入承诺；违约提高受损集团不满和反对组织。

## 7. 双重权力

进入双重权力后，各区域可分别由旧国家或替代机构控制五项职能：

| 职能 | 旧国家载体 | 替代载体 |
|---|---|---|
| 税收 | 户部、地方官、契约 | 革命税、会馆出资、村社征集 |
| 粮运 | 漕运、仓储、军需 | 地方粮委、工农组织、商人合同 |
| 司法 | 官府、都察院、刑部 | 地方议会、军事法庭、群众裁判 |
| 武装 | 卫所、新军、家丁 | 民兵、倒戈军、工人/农民武装 |
| 任命 | 皇帝、部院、官职继承 | 选举、推举、委员会或军令 |

每个职能控制 0—100。替代机构只控制武装而没有粮税，属于起义军；控制税粮而没有合法性和军队，容易被清剿；五项中至少三项超过旧国家，才可尝试全国主权宣告。

## 8. 革命战争不是普通夺地战争

战争胜负同时读取：

- 军队数量、质量、军工和补给；
- 区域阶级支持和地方组织；
- 城市、运河、港口、仓库和矿冶控制；
- 旧国家财政、外援和朝贡体系；
- 革命联盟纲领、纪律和内部信任；
- 镇压、报复和土地政策对占领区的反馈。

占领一县不等于治理一县。军队掠夺、强征或违背土地承诺会让地方支持快速转为抵抗。严格纪律需要可靠补给，否则只是按钮文本。

## 9. 反革命的资源

反革命不是随机刷出的负面派系。旧集团可以动用：

| 资源 | 载体 | 对策与代价 |
|---|---|---|
| 正统名分 | 宗室、礼制、地方士绅和宗教 | 新合法性、妥协或压制；过度压制伤害中立者 |
| 土地与粮食 | 地主、庄园、仓储和乡村网络 | 土地改革、征购、赎买；激进没收可能破坏供给 |
| 信用与资本 | 商人、债权、海外网络 | 债务重组、资本管制、合作；可能造成信用崩溃 |
| 官僚知识 | 文书、税册、法律和干部 | 留用、审查、培训替代；全面清洗导致治理瘫痪 |
| 军队和家丁 | 正规军、军镇、私人武装 | 分化、整编、缴械和新军；速度过快触发叛乱 |
| 地方中介 | 宗族、乡约、土司和会馆 | 自治契约、代表、替代组织；一刀切导致广泛反抗 |
| 外部支持 | 邻国、朝贡国、贸易伙伴和流亡集团 | 外交、封锁、让利或战争；损害主权和国内联盟 |

旧阶级的物质基础未改变时，即使其领袖被处决，新的代表仍会出现。

## 10. 夺权后的七项整合任务

革命/政变胜利后立刻进入整合，不给“革命成功”永久奖励。

### 10.1 粮食与城市供给

稳定征购、价格、运输和救济。失败会使城市劳动者和军队首先动摇。

### 10.2 税收与货币

决定延续旧税、发行信用、没收、借款或新税。过度依赖一次性没收会在数年后枯竭。

### 10.3 军队统一

整编倒戈军、民兵、地方武装和旧军。保留自治会威胁中央，立即缴械会触发盟友反叛。

### 10.4 官僚连续与清洗

留用旧官提高能力，却给反革命网络入口；全面清洗提高忠诚，却使文书、财政和司法瘫痪。

### 10.5 土地、债务与资本

兑现对农民、商人、工人和士绅的矛盾承诺。任何选择都会让联盟一部分失望。

### 10.6 地方与边疆契约

决定中央统一、自治、联邦、土司或民族/地域权利。革命核心区的制度不能无成本复制全国。

### 10.7 合法性与继承

确定君主、议会、委员会、军政领袖或选举的产生方式，以及紧急权力何时结束。没有继承规则的新制度会在领袖死亡时重新危机。

## 11. “热月”、官僚化与路线变形

革命联盟在整合中可能出现：

- 官僚掌握文书和预算，逐步排除群众组织；
- 军队以安全为由延长紧急权力；
- 商人以供给和信用换取产权寡头地位；
- 小农在获得土地后转向保护小产权，反对继续集体化；
- 工人组织要求管理权，与技术官僚和生产目标冲突；
- 地方自治抵抗中央统一税军。

这些不是固定“背叛事件”，而由控制资源和组织差异产生。游戏中称为整合路线变化，避免把所有制度化都道德化为单一结论。

## 12. 复辟的五种形态

| 复辟 | 恢复内容 | 保留变化 | 结果 |
|---|---|---|---|
| 王朝复辟 | 皇室、礼制和旧国号 | 可能保留新军、税制、工业 | 新旧混合君主制 |
| 士绅复辟 | 地权、地方调解和功名政治 | 部分市场法权和地方议会 | 地方寡头秩序 |
| 官僚热月 | 专业官僚和秩序压倒群众参与 | 公营工业、统一预算、部分社会政策 | 官僚发展国家 |
| 军事复辟 | 军队接管、停止政治动员 | 军工、土地承诺的部分内容 | 军政国家或军阀联合 |
| 商业寡头复辟 | 产权、债务和城市精英控制 | 议政形式、市场和工业 | 商业共和国/寡头制 |

复辟不会把变量重置到开局。土地已重新分配、军队已国家化、工业和组织已形成时，旧名号只能在新物质基础上重建。

## 13. 革命后的长期阶级变化

| 政策 | 短期 | 长期新矛盾 |
|---|---|---|
| 均田/限田 | 小农支持、地主反抗 | 土地再集中、继承碎片、资本不足 |
| 债务取消 | 解放负债者 | 信用冻结、资本和商人反对 |
| 国有工业 | 国家获得产能 | 官僚控制、预算和劳动参与冲突 |
| 工人管理 | 组织和忠诚提高 | 跨厂协调、投资与专业管理 |
| 地方自治 | 获得地域支持 | 全国税军、市场和权利标准不一 |
| 普遍征兵 | 军队社会基础扩大 | 军事化、退伍安置和政治权利诉求 |
| 扩大教育 | 干部和传播提高 | 新知识群体要求进一步参与 |

革命不会消灭矛盾，而是改变矛盾的位置、主体和制度表达。

## 14. 非一边倒的博弈规则

- 革命集团必须供给自己控制的城市和军队，不能无限依赖热情；
- 镇压不是纯失败：可能恢复生产和治安，也会积累记忆与地下组织；
- 妥协不是纯胜利：可避免内战，也可能冻结不平等或让旧集团重组；
- 土地改革提高支持，但处理过急可能破坏粮运、信用和中间小业主；
- 商人合作可解决供给，却会要求政治和产权承诺；
- 留用官僚提高执行，却带来旧网络和官僚自主；
- 军队倒戈决定短期胜负，也会要求长期预算、职位和政治权力；
- 外援提高胜率，却带来领土、贸易、债务或外交义务；
- 革命胜利者会分裂，失败者会学习，下一轮不会重复同一剧本。

## 15. CK3 实现

| 对象 | 载体 |
|---|---|
| 动员阶段 | situation phase parameter + class organization/grievance |
| 政治纲领 | alliance variable、character stance、law package |
| 合法请愿/罢市 | interaction、decision、county modifier、event |
| 地方接管 | faction、title modifier、special relation |
| 双重权力 | 区域锚点五项职能控制变量 |
| 革命战争 | custom CB + war contribution + occupation effects |
| 军队倒戈 | MAA/title transfer effect，严格限于相关军队 |
| 新政权 | government/law/succession package，不一次改写全部人物 |
| 整合任务 | story cycle 或 situation 子阶段 |
| 复辟 | faction/war/decision，按当前结构生成混合结果 |

### 15.1 战争与角色限制

- 不遍历所有世界人物选择革命者；只看相关政权重要人物、联盟成员、地方领主和军队持有者；
- 民众力量用区域变量和县级修正进入战争，不创建海量农民角色；
- 军队倒戈必须按饷源、指挥官、政治立场和地方支持判断，不随机转移全部兵士；
- 占领效果按区域季度结算，不按每支军队每日触发。

## 16. AI 决策

AI 是否支持革命/反革命：

```text
support_score = expected_class_gain
              + personal/family_survival
              + ideology_match
              + alliance_obligation
              + perceived_winner
              + grievance
              - repression_risk
              - property_loss
              - chaos_aversion
```

AI 不知道真实胜率，只根据情报和关系估计。机会主义者会晚加入领先方；信念型领袖更早承担风险；家族利益可使同一思想人物退缩。失败经历、亲属被杀、财产没收和承诺兑现都会留下长期记忆。

## 17. 玩家反馈

革命危机界面显示：

- 旧国家与替代联盟对税、粮、司法、武装、任命的控制；
- 各联盟纲领和不可兼容条款；
- 军队可靠性、主要将领和饷源；
- 核心区域的群众支持、供给和组织；
- 外部势力的支持与条件；
- 夺权后七项整合任务的准备度；
- 可能出现的妥协、镇压、碎片化和复辟路线。

玩家不能只看“革命进度 73%”；必须知道缺的是粮、军队、合法性还是组织。

## 18. 验收情景

### A：高不满、低组织

出现逃亡、抢粮和地方暴乱，但没有全国革命。军头或士绅可吸纳不满，证明人数与组织分离。

### B：城市革命、乡村中立

革命控制首都和工业城市，却因粮运、土地纲领和乡村组织不足陷入供给危机，被迫与小农或商人妥协。

### C：农民起义、官僚倒戈

地方均平运动得到改革官僚的税册和治理知识，从破坏性起义转成替代政府；若军队仍可靠忠于旧朝，战争依然艰难。

### D：新军政变

群众组织弱、军队可靠性崩溃，新军以救国为名夺权，进入军事发展或军阀路线，不错误标记为人民革命。

### E：革命胜利后的热月

旧官僚留用解决财政，逐渐排除群众组织；若仍兑现土地、工资和公共政策，形成官僚混合国家，而非简单“革命失败”。

### F：王朝复辟但社会未复旧

宗室重登皇位，却必须接受新军、统一预算、土地改革和议政机构，证明复辟不重置物质结构。

## 19. 验收清单

- [ ] 改革、改朝换代、暴乱、革命和革命胜利有明确不同定义；
- [ ] 动员从 M0 到 M6 逐级升级并留下组织经验；
- [ ] 革命需要国家危机、精英分裂、群众组织、纲领、治理和军队裂解；
- [ ] 意识形态由七个纲领轴组合，并受概念可用性限制；
- [ ] 至少三类能力结合后才形成可行革命联盟；
- [ ] 双重权力分别计算税、粮、司法、武装和任命；
- [ ] 占领不等于治理，军纪和供给影响地方支持；
- [ ] 反革命的名分、土地、信用、官僚、军队、地方和外援均有事实载体；
- [ ] 夺权后必须处理七项整合任务；
- [ ] 官僚化、军政化、寡头化和路线分裂由资源控制产生；
- [ ] 复辟保留已发生的土地、军队、工业和组织变化；
- [ ] AI 根据利益、关系、思想、风险和估计胜率选择立场；
- [ ] 革命战争不做每日全图扫描，不生成海量平民角色；
- [ ] 玩家能看见具体能力缺口，而非单一革命进度条。

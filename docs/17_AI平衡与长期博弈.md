# AI、平衡与长期博弈

> 文档性质：所有政治联盟、人物、国家与危机共用的 AI 决策和数值平衡规范  
> 性能、缓存、存档和测试预算见 [17A](17A_性能缓存存档与测试预算.md)。

## 1. AI 不是替玩家走历史路线

AI 的目标是让每个行动者根据自身物质利益、职位、思想、关系、风险和有限情报作出可解释决策。它不预知历史，也不读取玩家看不见的真实变量；它可以判断错误、背叛盟友、坚持失败路线，但这些错误必须来自信息、利益或性格，而不是纯随机。

```text
AI 决策 = 利益 + 生存 + 组织承诺 + 思想 + 关系 + 可行性 - 成本 - 风险
```

## 2. 四层 AI

### 2.1 人物层

决定任命、结盟、请托、调查、承诺、背叛和危机选项。读取人物职位、家族资产、关系、性格、功绩、影响力、事件记忆和最多三个利益来源。

### 2.2 政治联盟层

决定支持何种议程、投入多少资源、是否退出执政联盟、是否升级集体行动。联盟效用是成员利益的加权聚合，但领袖、组织规则和内部派别会造成偏移。

### 2.3 国家层

决定预算、战争、改革组合、工程、外交与危机优先级。国家 AI 不等同统治者个人：皇帝偏好必须与官僚执行、军队、财政和主导联盟共同计算。

### 2.4 社会结构层

没有“阶级 AI 角色”。年度调度器根据物质变化、组织度和不满产生压力、支持和行动机会；只有形成政治联盟和代表后，才进入人物/联盟 AI。

## 3. 信息边界

AI 与玩家使用相同信息质量：

- 统治者可见本国报告区间，不见真实隐田、地方支持和对手资产；
- 地方官知道本区更多，但有上报偏差；
- 商人更懂价格、信用和库存，不自动知道军队计划；
- 将领更懂军队、补给和敌情，不自动知道真实国库；
- 民众组织更懂生计和地方动员，不自动知道中枢密议；
- 情报、关系、职位和调查提高估计质量；
- AI 只能读取属于自己 scope 的估计缓存，不直接调用全局真实值作决策。

“困难 AI”可以更好利用已知信息、保存承诺和识别结构风险，但不能获得免费全知。

## 4. 通用效用函数

所有重要选择使用同一骨架，再按机制配置权重：

```text
utility = survival_gain
        + material_gain
        + office_or_state_gain
        + class_or_family_gain
        + ideology_match
        + alliance_obligation
        + relationship_gain
        + public_credit_gain
        + feasibility
        - direct_cost
        - implementation_risk
        - personal_and_family_risk
        - empowered_rival_risk
        - broken_commitment_cost
        - chaos_cost
```

每项先标准化到相近范围。禁止某一项使用远大于其他项的隐式数量级，导致其他因素永远无效。

### 4.1 物质收益

读取收入、土地、合同、俸饷、税负、工资、军功和产业控制；不是简单读角色黄金变化。

### 4.2 制度收益

读取职位、预算控制、任命权、代表权、地方自治和军队控制。长期制度收益可高于一次性金钱。

### 4.3 思想匹配

思想决定愿意为某方案承担多少短期损失，但设上限。强信念人物仍会考虑家族生存和执行可能。

### 4.4 关系和承诺

恩主、亲属、朋友、对手、联盟和已签承诺影响选择。承诺不是普通好感；违约单独计成本，并进入未来记忆。

### 4.5 风险感知

AI 读取估计胜率区间、镇压能力、军队可靠性、债务和替代方案。勇敢/野心提高可承受风险，谨慎/安于现状降低，但不改变事实条件。

## 5. bounded rationality：有限理性

AI 不在所有可能方案中求全局最优，只在当前阶段生成 2—5 个可行方案：

1. 过滤硬不可行；
2. 按角色/联盟议程生成候选；
3. 使用估计信息评分；
4. 给最高若干方案小幅随机扰动；
5. 选择并锁定一段时间；
6. 只有重大新信息、领袖更替或失败阈值才重评。

这能产生合理差异，同时避免每季度在几十条路线间抖动。

## 6. 路线惯性与转向

| 状态 | 惯性期 | 可提前打破 |
|---|---:|---|
| 普通政策 | 2—3 年 | 预算中止、法令失效、领袖死亡 |
| 重大改革 | 4—8 年 | 国家危机、联盟瓦解、严重地方抗拒 |
| 工业项目 | 至少一个项目阶段 | 破产、领土丢失、战争紧急征用 |
| 战争战略 | 一个战季/年度 | 首都受威胁、盟友退出、军粮崩溃 |
| 革命/反革命阵营 | 1—3 年 | 生命威胁、重大背叛、阵营失去治理能力 |

转向需要支付沉没成本、信誉和联盟成本。AI 不会仅因评分差 1 点取消前任十年工程。

## 7. AI 记忆和学习

只保存高价值摘要，不记录每个事件：

| 记忆 | 作用 | 建议期限 |
|---|---|---:|
| 某联盟兑现/背叛承诺 | 未来谈判和结盟 | 10—30 年或人物终身 |
| 某地区改革贯彻/造假/反抗 | 选择执行者和试点 | 10—20 年 |
| 某军队足饷/兵变/倒戈 | 军事信任和预算 | 10—25 年 |
| 某融资方式违约 | 商业信用和利息 | 10—20 年 |
| 镇压成功/失败 | 强硬路线信心 | 5—15 年 |
| 工程成功/烂尾 | 技术路线与投资 | 10—25 年 |
| 灾荒赈济表现 | 国家/地方合法性 | 10—20 年 |

记忆影响权重，不硬锁选择。继任者继承国家和家族记忆的一部分，个人仇恨不全部继承。

## 8. 政治联盟 AI

### 8.1 联盟内部效用

```text
alliance_utility
  = leader_weight × leader_utility
  + major_factions_weighted_utility
  + common_program_match
  + organizational_survival
  - internal_split
```

强组织联盟更能约束领袖；个人恩庇联盟更受领袖性格影响。联盟可在政策投票前分裂、要求修正或以不参加代替公开反对。

### 8.2 支持强度

| 强度 | 行为 |
|---|---|
| 认可 | 口头支持，不投入资源 |
| 合作 | 提供影响力、执行者或小额财政 |
| 承诺 | 把组织和信誉绑定方案 |
| 动员 | 发动成员、舆论、市场或军队 |
| 生死斗争 | 退出常规政治，准备政变/革命/内战 |

支持不是二元布尔。玩家需要辨别谁只赞同、谁愿承担成本。

## 9. 国家 AI 的议程预算

每个国家同时只能维持有限主议程：

| 类型 | 标准上限 |
|---|---:|
| 主改革 | 2 |
| 大型工程 | 3，其中全国级最多 1 |
| 全国重大危机 | 2 |
| 区域重大危机 | 2 |
| 主动战争目标 | 1 个主战 + 1 个机会战 |
| 全国政治运动重点 | 2 |

超过上限时，AI 必须暂停、降级或拒绝新项目。玩家也受相同“行政注意力”约束，不靠 UI 同时开十项改革。

## 10. 危机优先级

```text
priority = existential_threat
         + time_pressure
         + affected_core_weight
         + coalition_salience
         + cascade_risk
         - current_response_progress
```

生存威胁、首都粮供、主力军欠饷和继承通常高于长期工业项目；但 AI 不能永久用“紧急”逃避维护，连续紧急状态会提高军政化、债务和社会不满。

## 11. 改革 AI

AI 先识别结构问题，再评路线：

```text
reform_score = problem_relevance
             + coalition_support
             + leader_program
             + regional_fit
             + expected_state_capacity
             - fiscal_gap
             - personnel_gap
             - local_resistance
             - rival_empowerment
```

- 官僚整饬 AI 偏好信息、预算、考成和统一规则；
- 宫廷 AI 偏好可直接控制的财源和代理；
- 商业 AI 偏好合同、市场、信用和基础设施；
- 军事 AI 偏好足饷、军工、战略交通和紧急权力；
- 民生 AI 偏好生计、土地、救济和参与；
- 地方秩序 AI 偏好低中央成本、地方裁量和地权。

偏好不是固定脚本。严重战争可使商业联盟支持军工，灾荒可使士绅支持减税，国家掠夺可使商人与劳动者临时联合。

## 12. 战争与外交 AI

### 12.1 战争成本必须完整

AI 估算：军队质量、补给、工期、国库、边疆信用、国内反对、战后驻军和机会成本。只比较敌我兵力会导致无限战争。

### 12.2 朝贡和属国

宗主要求贡赋、出兵或政治干预时，属国 AI 读取负担、保护信用、自主、替代宗主和国内集团。弱小不等于自动接受；拒绝可换来惩罚、重新谈判或寻找联盟。

### 12.3 外部挑战者

挑战者根据内部统一、边贸、盟友、天朝危机和补给扩张，不按年份获得免费军队。天朝 AI 会在分化、贸易、缓冲、改革和战争间选择。

## 13. 平衡目标

### 13.1 没有永久最优路线

每条主要路线至少包含：

- 三项独特能力；
- 三项结构性脆弱；
- 两种可被对手利用的反制；
- 一项继承/长期维持问题；
- 一种失败后仍可继续玩的变形结果。

### 13.2 短期收益与长期权力

借商款能救军费，但增强债权人；用将领私饷能保战线，但增强军头；士绅协办能快速清丈，但可能保护隐田；群众组织能高效赈济，也会要求代表。所有“便宜解决”都把未来权力给某人。

### 13.3 增长与分配分开

总产能、国家收入、平均生计、土地分配、工资、组织和政治参与分别计算。产能增长不能自动降低不满，平等政策也不能自动增加工业能力。

### 13.4 能力与控制分开

国家拥有军工能力不等于皇帝控制军工；区域拥有商业财富不等于中央能征税；民众人数多不等于能动员；官僚制度强不等于不被地方利益俘获。

## 14. 内生反制而非隐藏橡皮筋

领先者受到的反制来自结构：

| 领先方式 | 内生反制 |
|---|---|
| 皇权过强 | 官僚消极、地方隐匿、继承危机、公私财政冲突 |
| 官僚过强 | 程序阻塞、信息垄断、指标主义和政治排斥 |
| 士绅过强 | 国家税军弱、兼并、地方割裂和民众不满 |
| 商人过强 | 垄断、信用危机、工资冲突和政治俘获 |
| 军队过强 | 预算吞噬、军头化、政变和永久战争需求 |
| 民众运动过强 | 联盟碎片、供给、专业治理和地方协调压力 |
| 国家工业过强 | 官营低效、军工偏向、债务和劳动冲突 |

不使用“玩家太强所以随机 -50% 收入”或给弱方免费军队的隐藏难度修正。

## 15. 数值尺度

核心结构值使用 0—100，但大多数变化控制在：

| 变化 | 幅度建议 |
|---|---:|
| 普通事件 | ±0.5—2 |
| 政策年度效果 | ±1—3/年，受执行度缩放 |
| 重大危机结算 | ±3—10 |
| 革命/国家断裂 | ±8—20，但分多年落地 |
| 自然回归 | 每年向结构基线 0.2—1 |

单一事件不得直接把全国阶级力量从 20 变 80。重大转型也通过区域、制度和组织逐步写入。

### 15.1 阈值带

| 区间 | 语义 |
|---|---|
| 0—19 | 边缘/几乎无能力 |
| 20—39 | 弱但存在 |
| 40—59 | 可参与常规政治 |
| 60—79 | 强，能否决或主导部分议程 |
| 80—100 | 霸权/危机性集中，触发强反制 |

进入阈值与退出阈值错开 10—15，避免抖动。

## 16. 难度设置

| 项目 | 休闲 | 标准 | 困难 |
|---|---|---|---|
| 提示 | 更明确赢家、输家和风险 | 标准解释 | 不隐藏事实，只减少预测精度 |
| AI 规划 | 较短视、较少记忆 | 正常 | 更好保存预算和路线记忆 |
| 信息质量 | 玩家报告较准 | 与 AI 相同 | 利益集团更会隐匿，玩家与 AI 同受影响 |
| 危机时间 | 较宽 | 标准 | 截止期更紧但不额外造危机 |
| 违约成本 | 较低 | 标准 | 组织记忆更长 |
| 免费资源 | 无 | 无 | 无 |

难度不改变基础战斗伤害、阶级“忠诚税”或事件暗骰结果。

## 17. 反雪球设计

- 权力集中提高执行，也提高被排斥集团结盟收益；
- 高税收能力扩大国库，也扩大对预算分配的政治争夺；
- 高商业化扩大财富，也提高信用和外贸冲击；
- 高工业化扩大军力，也提高粮食、能源、工资和管理需求；
- 战争胜利提高合法性和军功集团，战后兑现成本随之上升；
- 革命胜利扩大参与，也必须处理供给、军队和官僚连续；
- 稳定不是永久奖励，长期压制矛盾会提高不可见压力。

## 18. 玩家失败仍可玩

失败不只结束游戏：

- 改革失败可转为地方试点、地下纲领或下一代记忆；
- 战败可触发财政重组、军队改革、属国离心或革命窗口；
- 被迫妥协会改变路线，而非清空进度；
- 革命失败可形成流亡、地方自治、军阀或后续温和改革；
- 复辟保留部分土地、工业、军队和议政变化；
- 国家分裂可在区域政权继续，形成再统一或多中心长期局。

只有玩家角色失去所有合法可玩继承且 CK3 本体判定结束时才真正结束。

## 19. AI 可解释性

重要 AI 决策在调试和必要 UI 中保存前三项正权重、前三项负权重：

```text
支持国家军器局：
+ 外部威胁高
+ 家族拥有技术家产
+ 执政联盟承诺军工预算
- 国库债务严重
- 该人物的商业合同将被官营取代
- 执行官是政治对手
```

正式 UI 不展示全部公式，但用自然语言解释立场。调试模式可显示标准化数值。

## 20. 验收情景

- [ ] 同一士绅人物能因职位、地产和思想组合支持或反对清丈；
- [ ] AI 不读取真实隐田、敌军计划和群众支持，只用情报估计；
- [ ] 重大改革具有 4—8 年路线惯性，不能因微小分差反复取消；
- [ ] 兑现/背叛、兵变、工程和赈济形成长期记忆；
- [ ] 联盟支持有认可、合作、承诺、动员和生死斗争五级；
- [ ] 国家同一时间受主改革、工程、危机和战争注意力上限；
- [ ] 所有主要路线有能力、脆弱、反制、继承问题和失败变形；
- [ ] 增长、分配、能力与控制分别计算；
- [ ] 领先者反制来自结构，不使用隐藏橡皮筋；
- [ ] 难度提高规划和信息对抗，不给 AI 免费资源；
- [ ] 普通事件不造成全国结构巨变，阶段阈值有滞后；
- [ ] 失败、妥协、分裂和复辟都有可继续玩的结果；
- [ ] 重要 AI 选择能输出主要正负理由。

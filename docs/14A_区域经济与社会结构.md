# 区域经济与社会结构

> 文档性质：阶级模型的空间层、生产关系数据结构和性能方案  
> 关联文档：[14 阶级力量与生产方式模型](14_阶级力量与生产方式模型.md)、[11A 县域赋役土地矿产与基础设施](11A_县域赋役土地矿产与基础设施.md)、[11B 私产地产行商与市场](11B_私产地产行商与市场.md)、[11C 人口控制与社会人口抽象](11C_人口控制与社会人口抽象.md)。

## 1. 设计目标

全国平均值会掩盖真实博弈：江南工商业、北方边防、京畿供给、东南海贸、湖广粮区、西南土司和矿冶地区面对同一政策时，收益、阻力和阶级联盟不同。新 Mod 使用少量功能区域表现差异，但不模拟每个村庄、人口个体或逐件商品。

区域系统回答五个问题：

1. 主要靠什么生产与交换；
2. 谁控制土地、劳动、资本和国家入口；
3. 国家如何征税、征役、供军和提供公共品；
4. 哪些冲击会改变人口和阶级构成；
5. 为什么同一改革在不同区域产生不同结果。

## 2. 八个区域结构轴

每个区域锚点只保存八项 0—100 核心轴，另有两项派生指标。这样既能表达生产关系，又避免“八阶级 × 七指标 × 每县”的变量爆炸。

| 轴 | 0 端 | 100 端 | 主要来源 |
|---|---|---|---|
| `land_concentration` | 土地分散、自耕占优 | 大地产与依附租佃占优 | 土地兼并、清丈、赎买、战争和债务 |
| `labor_coercion` | 自由迁徙和契约劳动 | 户籍、役法、债役、奴仆和强制征发 | 役制、军户、工匠义务、地方权势 |
| `commercialization` | 自给和地方交换 | 市场依赖、长途贸易和现金税 | 城市、道路、海贸、税制和商品生产 |
| `wage_labor` | 家庭/依附劳动 | 稳定工资依赖和劳动力市场 | 工场、失地、城市化、国家雇工 |
| `industrial_capacity` | 家庭手工业 | 集中工具、能源、技术与标准化生产 | 作坊、矿冶、军器局、教育和机器 |
| `state_direct_control` | 地方中介和私人支配 | 官府直接登记、征收、任命和生产 | 清丈、流官、官营机构、驻军 |
| `transport_integration` | 孤立、季节性运输 | 稳定水陆网和全国市场 | 运河、道路、港口、仓储和治安 |
| `extraction_pressure` | 轻且可预期 | 税、租、债、役和非常摊派沉重 | 战争、债务、官府与地主政策 |

派生指标：

```text
subsistence_security
  = 粮食基础 + 水利仓储 + 实际收入 + 救济能力
  - 灾害 - 价格波动 - 总提取

social_mobility
  = 教育与职业入口 + 迁徙自由 + 市场机会
  - 身份强制 - 地产封闭 - 债务依附
```

派生值每年缓存，不再另建复杂库存。

## 3. 七种生产关系模块

区域不是只能选一种生产方式，而是由若干模块混合。主导模块用于 UI，实际效果由八轴连续值决定。

| 模块 | 土地/工具所有 | 劳动组织 | 产品去向 | 典型矛盾 |
|---|---|---|---|---|
| 家庭自耕 | 小农家庭 | 家庭劳动与村社互助 | 自用 + 少量市场 | 税役、灾害与债务侵蚀 |
| 地主租佃 | 地主/士绅 | 佃户、分成或定额租 | 地租与市场并存 | 地租、失地、债役和抗租 |
| 户籍徭役 | 国家登记身份 | 无偿/折纳役与世袭义务 | 国家工程、军役和贡纳 | 逃役、冒籍、中介盘剥 |
| 军户屯田 | 国家/军镇名义控制 | 军户家庭、屯田和军役 | 军粮、边市和饷银 | 土地侵占、役重、军职私有化 |
| 小商品生产 | 家庭/作坊自有工具 | 家庭、学徒与少量雇工 | 区域市场 | 原料、信用、税卡和商人压价 |
| 包买商—分散工场 | 商人掌握原料/订单 | 分散家庭作坊 | 长途和外贸市场 | 债务、质量控制、价格支配 |
| 集中工场/国家工业 | 私人资本或国家掌握设备 | 稳定雇工、技术管理、纪律 | 大宗市场、军队和工程 | 工资、控制权、垄断、官僚低效 |

### 3.1 “再工业化”定义

本项目中的再工业化不是从零发明手工业，而是让已经存在的矿冶、纺织、造船、军器、陶瓷、造纸等生产摆脱周期性崩坏和身份役制限制，形成可持续的资本、能源、技术、劳动、运输和需求闭环。

区域进入再工业化候选状态，至少需要：

```text
industrial_capacity ≥ 中等
transport_integration ≥ 中等
commercialization ≥ 中等
可持续需求存在（市场、军队或国家工程）
技术人员和雇工来源存在
产权/官营治理至少一种可执行
```

满足条件只解锁路线，不自动增长。土地收益过高、特许寻租、战争破坏、信用危机、劳动力强制和能源瓶颈都可能让资本回流地产或贸易。

## 4. 功能区域，而非固定优劣省份

建议在地图适配时建立 8 个左右稳定功能区。名称与边界需根据实际“溥天之下”地图头衔和 geographical region 复核，下面只定义玩法角色。

| 功能区 | 初始倾向 | 国家依赖 | 主要矛盾 | 潜在转型 |
|---|---|---|---|---|
| 京畿—中枢供给圈 | 官府、宫廷消费、仓储和运输集中 | 漕粮、财政转运、军队保护 | 内廷与公共财政、消费与生产脱节 | 行政工业、军工、全国金融中心 |
| 北方九边—辽东 | 军户、军镇、边市和高军费 | 饷银、粮运、马匹、军工 | 欠饷、军头化、边贸与敌对关系 | 军事消费市场、国家军工或割据经济 |
| 西北军粮—农牧交错区 | 低运输、屯田、灾害敏感 | 边防和水利 | 军役、土地退化、迁徙与族群契约 | 灌溉农业、边贸、军事共同体 |
| 江南工商—漕粮区 | 高商业、手工业、地租与城市 | 税粮、纺织、信用和技术 | 地租、包买商、税权和城市劳动 | 分散工场、集中工场、金融资本 |
| 东南海贸区 | 港口、走私/合法贸易、商帮 | 银流、海防、关税 | 海禁与开放、特许与自由贸易、沿海军事 | 造船、海贸资本、外向工业 |
| 中原—湖广粮食与移民区 | 农业扩张、宗族、市场和水运混合 | 粮食、兵员、长江运输 | 土地兼并、宗藩、灾荒与流民 | 商品农业、内河工业、民众运动 |
| 四川盆地—上游资源区 | 相对完整农业圈、盐矿和长途运输瓶颈 | 盐、粮、矿产和西南通道 | 地方财政、运输成本、外来军政 | 盐业资本、军工和上游市场整合 |
| 西南土司—矿冶边疆 | 土司契约、矿产、移民与多重权利 | 矿税、战略通道和军事盟约 | 改土归流、土地掠夺、劳役和文化自治 | 契约整合、殖民采掘或地方联合体 |

这些是初始倾向，不是永久加成。江南可以因战争和苛税去工业化；边疆可以因军费、市场和交通成长；京畿可以在财政崩溃后失去供给；西南矿冶可以成为国家工业节点，也可以退化为强制采掘区。

## 5. 区域到阶级的派生规则

区域轴每年为全国阶级物质基础提供增量。以下为方向逻辑，最终权重在平衡阶段标定。

| 区域变化 | 受益 | 受损/转化 |
|---|---|---|
| 土地集中上升 | C2 经济基础、部分 C5 债权 | C7 转向 C8；国家税基可见度可能下降 |
| 劳动强制上升 | C1/C2/C4 的短期控制 | C7/C8 不满、逃亡；长期生产率和流动下降 |
| 商业化上升 | C5/C6；部分 C7 副业 | 价格波动暴露；依赖现金者风险上升 |
| 工资劳动上升 | C6 生产规模、C8 人口基础 | C7 比重下降；劳动冲突和组织条件上升 |
| 工业能力上升 | C4/C5/C6/C8 的不同分支 | 取决于所有制；可能挤压传统作坊 |
| 国家直控上升 | C3、部分 C1/C4 | 地方 C2 和土司中介；若低薪则腐败上升 |
| 交通整合上升 | C3/C5/C6、粮食安全 | 疫病、军队和危机传播也加快 |
| 提取压力上升 | 收入获得者短期受益 | C7/C8 生计下降；逃税、抗租、罢市和兵变 |

### 5.1 所有制决定工业化的阶级后果

同样增加 10 点工业能力，后果取决于控制者：

| 所有制 | 产能 | 利润/剩余去向 | 政治后果 |
|---|---|---|---|
| 官营部院 | 稳定但依赖预算和官僚能力 | 国库、官僚承包与腐败损耗 | C3/C4 增强，可能压制 C5 |
| 宫廷皇产 | 可绕开公共预算 | 内帑、宦官和皇商 | C1 增强，公共财政冲突 |
| 商人资本 | 对市场和信用敏感 | C5、投资者和经理 | 要求法权，可能压低工价 |
| 作坊联合 | 中等规模、技术自治 | C6 与成员 | 组织分散，融资和标准化困难 |
| 军镇控制 | 战时有效 | 军费、将领和军商 | C4 强化，中央失控风险 |
| 劳动合作/公社 | 分配较平但治理要求高 | C8/地方公共机构 | 需要组织、技术和市场协调 |

## 6. 政策执行的区域公式

每项全国政策为各区域分别结算，不用一个全国成功率覆盖所有地方。

```text
regional_execution
  = national_legal_strength
  × local_state_control
  × implementing_official_capacity
  × supporting_class_power
  × fiscal_supply
  × information_quality
  ÷ organized_resistance
```

结果分五档：

| 档位 | 含义 | 后果 |
|---|---|---|
| 贯彻 | 达到政策目标且形成新常规 | 结构轴稳定变化，支持者组织上升 |
| 妥协 | 目标部分实现，交换特权 | 效果较低，产生地方版本和承诺 |
| 形式执行 | 册报完成但实际关系不变 | 短期合法性，后续审计爆雷 |
| 反向转嫁 | 成本转给弱者 | 表面财政改善，生计和不满恶化 |
| 公开抗拒 | 地方拒绝或冲突 | 联盟对抗、罢市、民变或军事介入 |

### 6.1 清丈例子

- 高国家直控、高官僚能力、地方小业主支持：隐田被发现，税负更均，C3/C7 获益；
- 低信息、高士绅组织：允许民自报，形成形式执行；
- 高指标压力、低监督：官员把坡地、湖荡、屋基等虚增为田，形成反向转嫁；
- 强士绅与地域舆论合流：清丈从税政争议升级为身份、地域和名分冲突；
- 后续未维护册籍：数年后土地关系重新脱离账面。

这直接把《万历明君》强调的执行变形转成可重复机制，而不是一次性剧情。

## 7. 人口转化与迁徙

不保存精确人口，只保存比例趋势和压力。主要流动：

```text
自耕小农 --债务/灾害/兼并--> 佃户
自耕/佃户 --季节市场--> 临时雇工
佃户/流民 --城市需求--> 稳定工资劳动
工匠 --资本与订单--> 作坊主或工场雇工
商人 --购地/功名--> 士绅地主
士绅/官僚 --商业投资--> 士商/工业资本
军户 --逃役/募兵--> 农民、雇佣军或军头依附
```

迁徙推力：灾害、失地、提取、战争、身份强制和失业。拉力：工资差、土地、治安、救济、工程、军队招募和交通。国家可以限制迁徙，但高压只会把合法迁徙变成逃户、走私和地下劳动力。

## 8. 冲击传播

| 冲击 | 首发轴 | 传播渠道 | 可能结果 |
|---|---|---|---|
| 灾荒 | 生计安全下降 | 粮价、迁徙、债务、税役 | 失地、流民、赈济联盟或民变 |
| 银/信用紧缩 | 商业与财政结算受阻 | 税银、借贷、军饷 | 破产、欠饷、价格错配 |
| 战争 | 提取和军需上升 | 运输、征兵、军工、难民 | 边区繁荣或毁灭、军头化 |
| 海贸扩张 | 商业化和银流上升 | 港口、商帮、原料订单 | 工场增长、走私转正或特许垄断 |
| 交通工程 | 运输整合上升 | 市场、军队、疫病 | 专业化、粮安改善或征发冲突 |
| 技术突破 | 工业潜力上升 | 技工、资本、订单和制度 | 有条件扩散；不满足条件则停留样品 |
| 镇压 | 组织短降、不满上升 | 记忆、流亡、秘密结社 | 恐惧稳定或激进化 |

交通越整合，收益与危机传播都越快。高商业化区域不会获得纯增益，而要面对外部价格、信用、疫病和战争封锁。

## 9. 数据载体与初始化

### 9.1 区域锚点

每个功能区选择一个不会随统治者死亡而消失的 landed title 作为数据锚点。锚点只保存结构变量，不表示该头衔持有者自动拥有整个区域。区域边界用 `geographical_region` 或预生成 title 列表判断。

```text
title:<region_anchor>.var:minghm_land_concentration
title:<region_anchor>.var:minghm_labor_coercion
...
```

具体 title ID 在地图适配和 `18` 数据字典中冻结。若锚点头衔不存在或地图模组改变，初始化应中止该区域并写错误日志，不随机选择替代头衔。

### 9.2 初始化来源

开局一次性读取：

- 区域内开发度、城市/城堡/寺庙比例；
- 市场、港口、矿产、农田、军工和交通建筑；
- 天朝省、军政头衔、土司/边疆政府与主要契约；
- 历史剧本给出的固定校正值。

初始化把地图信息压缩成八轴。之后主要由事件、政策、工程和年度漂移改变，不每月重新扫描全区。

## 10. 更新算法

### 10.1 即时增量

建筑完工、工程、战争、灾害、政策和重大事件直接调用统一 effect 修改相关轴，并记录原因码：

```text
minghm_change_region_structure_effect = {
  REGION = <anchor>
  AXIS = industrial_capacity
  VALUE = 3
  REASON = state_arsenal_completed
}
```

### 10.2 年度结算

每年每区一次：

1. 应用自然回归和持续政策；
2. 结算生计安全与社会流动；
3. 推导人口转化趋势；
4. 把区域贡献汇入全国阶级物质基础；
5. 选择最多一个区域结构事件；
6. 缓存 UI 文本和主要变化原因。

### 10.3 五年校准

每五年才抽样检查地图地产、发展和建筑是否与缓存严重偏离。优先抽取首府、最高发展、最低发展和随机县，不全量遍历所有 barony。偏差超过阈值时渐进修正，避免一帧跳变。

## 11. 全国聚合

区域对全国的权重不只按面积。建议：

```text
regional_weight
  = 0.35 × development_share
  + 0.20 × taxable_base_share
  + 0.15 × urban/industrial_share
  + 0.15 × population_proxy_share
  + 0.15 × strategic_weight
```

战略权重用于京畿、边防、运河和关键港口，但设上限，防止一处边镇决定全国社会构成。全国值是政治决策尺度；区域值决定执行和局部危机，两者不能互相替代。

## 12. 玩家界面

全国页：区域地图着色只提供五种可切换视图——土地集中、商业化、工业能力、国家直控、生计安全。

区域提示显示：

- 主导与次要生产关系；
- 八轴中最高/最低的三项；
- 最近五年主要变化；
- 当前改革执行档位；
- 活跃阶级与可能冲突；
- 可用工程和政策。

不显示虚构精确人口、GDP 或吨位。若需要“产出”，使用相对能力指数，并明确它是游戏抽象。

## 13. 平衡原则

- 高土地集中可提高短期可征租剩余，但降低小农韧性和国家可见税基；
- 高商业化提高交换效率，也提高价格和信用冲击；
- 高国家直控提高执行力，也增加官僚成本与滥权风险；
- 高劳动强制提高短期动员，降低长期生产率、迁徙合法性和忠诚；
- 高工资劳动增加工业弹性，也增加失业与集体议价；
- 高运输整合同时强化粮运、征税、军队、疫情和叛乱传播；
- 工业能力只有与能源、需求、技术、交通和组织匹配才转为持续增长；
- 任何区域都能转型，也都能退化，不设置永久“先进/落后民族”修正。

## 14. 验收情景

- [ ] 江南高商业开局不自动获得最高国家税收，地方中介和隐匿会影响实现；

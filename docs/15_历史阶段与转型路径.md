# 历史阶段与转型路径

> 文档性质：社会转型 situation 的阶段状态机与多路线总图  
> 数据底座：[14 阶级力量与生产方式模型](14_阶级力量与生产方式模型.md)、[14A 区域经济与社会结构](14A_区域经济与社会结构.md)  
> 专项设计：改革、再工业化与资本形成见 [15A](15A_改革再工业化与资本形成.md)；革命、反革命与复辟见 [15B](15B_革命反革命与复辟.md)。

## 1. 核心原则

历史阶段不是年份、朝代名、单项科技或不可逆进度条，而是生产关系、国家形式、动员方式和主导思想暂时形成的稳定组合。它会因内部矛盾、外部冲击、改革成败和阶级联盟变化而转型，也可能停滞、回退、复辟或形成混合制度。

```text
阶段 = 物质结构 + 国家制度 + 动员方式 + 意识形态霸权
```

玩家看见的“局势阶段”描述当前主要矛盾；底层四个制度层决定实际规则。这样可以避免把不同路径都压进一条“现代化值”。

## 2. 四个并行制度层

### 2.1 生产制度

| 档位 | 主导关系 | 关键能力 | 内在矛盾 |
|---|---|---|---|
| 农业—身份役制 | 家庭农业、户籍义务、地租、徭役和军户 | 低现金需求、地方自给、身份动员 | 逃役、兼并、身份固化、国家看不见资源 |
| 商业化农业 | 商品作物、银税、地租货币化和长途交换 | 市场、信用和运输 | 价格冲击、债务失地、区域分化 |
| 分散原工业 | 家庭作坊、包买商、官府订单和专业化地区 | 技术工匠、订单、商帮 | 商人压价、资本投地、规模与能源限制 |
| 集中工场 | 稳定雇工、工具集中、标准化、经理控制 | 资本、劳动市场、技术和大宗需求 | 工资冲突、失业、垄断与组织化 |
| 机械/能源工业 | 动力、机器、矿冶、交通和大规模再投资 | 科研、能源、工程、全国市场 | 资本集中、环境与粮食约束、劳动政治 |
| 混合/去工业化 | 多种关系并存或生产链退化 | 取决于区域 | 战争破坏、市场断裂、技术流失与复旧 |

### 2.2 国家形式

| 类型 | 资源控制 | 官僚与军队 | 优势 | 风险 |
|---|---|---|---|---|
| 宫廷家产国家 | 公私财政边界弱，内廷直接支配 | 依赖恩宠、内官和私人代理 | 决策快、能绕开官僚阻挠 | 任意征敛、继承断裂、制度信任低 |
| 官僚财政国家 | 预算、考成、册籍和职位规则化 | 专业官僚与国家化军队 | 可持续征收和执行 | 刚性、信息造假、官僚自我保护 |
| 地方士绅共治 | 中央通过地方精英征收和治理 | 地方调解、宗族和绅权强 | 成本低、地方知识丰富 | 隐田、地方俘获、区域割裂 |
| 商业法权国家 | 契约、信用、商税和投资代表增强 | 官商合作、城市自治或议政 | 资本和市场动员强 | 寡头化、公共责任不足、城乡冲突 |
| 军政动员国家 | 战争和工业目标支配预算 | 军队、军工与行政融合 | 紧急动员和战略建设 | 军头化、民生挤压、长期军事专断 |
| 大众参与国家 | 地方议会、工农组织和公共机构进入治理 | 军队政治化或民兵化 | 高参与和再分配能力 | 联盟碎片化、专业治理与供给危机 |
| 碎片化多中心 | 中央失去税军，地方各自建制 | 军阀、城市、宗族、土司并立 | 局部实验和自治 | 战争经济、交易成本、统一市场崩解 |

### 2.3 动员方式

| 方式 | 国家取得资源的方法 | 谁获得力量 | 长期后果 |
|---|---|---|---|
| 身份义务 | 军户、匠户、里甲、徭役和贡纳 | 地方中介、身份管理者 | 现金成本低但逃役、僵化和低效率 |
| 税收—官僚 | 统一税则、预算和专业机构 | 财政官僚、受薪军队 | 国家能力强，需稳定税基和监督 |
| 市场—合同 | 采购、承包、借款、特许和工资 | 商人、承包商、工场主 | 灵活高效，也会俘获国家和扩大债务 |
| 军事征发 | 战时加派、征兵、军管和强制生产 | 将领、军需集团 | 短期爆发力，长期破坏生产和合法性 |
| 国民动员 | 普遍税役、征兵、教育和政治宣传 | 中央国家、专业组织 | 扩大参与和能力，也扩大国家侵入生活 |
| 群众自组织 | 村社、会馆、工场、议会、互助与民兵 | 工农、市民和地方组织 | 高组织潜力，协调成本与派系冲突上升 |

### 2.4 意识形态霸权

意识形态不是独立魔法，而是把现实利益解释为“合理秩序”的语言。一个阶段可以同时存在多种纲领，但通常有一种主导合法性。

| 纲领 | 合法性来源 | 偏好的国家/经济 | 可能支持者 |
|---|---|---|---|
| 皇统—民本 | 天命、祖制、赈民和君主责任 | 宫廷集权或仁政修复 | 皇室、部分官僚、小农 |
| 功绩官僚主义 | 科举、考成、统一规则和专业治理 | 官僚财政国家 | 职业官僚、改革士绅 |
| 地方名分主义 | 宗族、乡约、产权和地方道德权威 | 士绅共治 | 地主士绅、小业主 |
| 商业法权主义 | 契约、信用、财产权和贸易富国 | 商业法权国家 | 商人、工场主、城市精英 |
| 国家富强主义 | 军工、技术、统一、国防和生产力 | 官僚或军政发展国家 | 官僚、军队、技术人员 |
| 农民均平/民生主义 | 生存权、减租、均田、赈济和共同体 | 再分配国家或乡村自治 | 小农、佃户、基层士人 |
| 市民/人民主权 | 纳税、服役与政治代表相联系 | 议政、共和国或混合政体 | 城市居民、商人、部分官僚和劳动者 |
| 劳动共同主义 | 劳动创造、共同管理、工资和社会保障 | 合作、公营或劳动参与制度 | 雇工、工匠、激进知识分子 |
| 军事救国主义 | 战争生存、纪律、牺牲与统一 | 军政动员国家 | 军队、难民、边区社会 |

这些名称是游戏设计概念，最终本地化应根据时代和路线使用当时可理解的政治语言。现代“主义”不能从未来知识按钮瞬间普及；它需要印刷、教育、翻译、共同斗争和组织传播。

## 3. 玩家可见的十个局势阶段

局势阶段描述“矛盾如何运动”，不是给国家贴永久文明等级。

### P0：旧秩序再生产

国家财政、地方中介、土地关系和身份动员仍能维持。矛盾存在但未迫使统治联盟重组。

- 主要玩法：积累财政、建立信息、培养替代干部、选择局部试点；
- 危险：玩家误以为稳定等于无问题，结构压力在后台累积；
- 出口：商业化、战争、财政缺口、灾害或权力继承使旧规则不再匹配现实。

### P1：结构张力上升

商业化、土地集中、军费、城市化、身份逃逸或技术变化加速，制度仍按旧方式分配成本。

- 主要矛盾：新财富与旧政治入口、现金经济与固定税制、军费与小农再生产；
- 主要玩法：选择谁承担调整成本，建立改革或反改革联盟；
- 出口：缓和回 P0；改革议程进入 P2；危机失控进 P4。

### P2：改革议程竞争

多个联盟提出不同解决方案，争夺皇帝、内阁、部院、军队和舆论。

- 主要玩法：议程设置、官职任命、预算、试点、政治交易和反制；
- 不是所有改革都进步，也不保证通过；
- 出口：形成执行联盟进 P3；联盟瓦解退 P1；危机抢先爆发进 P4。

### P3：制度重构与试点扩散

法令已通过，但地方执行、财政支持和新制度再生产尚未稳定。

- 主要玩法：清丈、培训、审计、工程、补偿、镇压或妥协；
- 最容易出现“形式成功、实质变形”；
- 出口：制度稳定进 P7；生产和组织发生跃迁进 P6；反弹进 P5；全面危机进 P4。

### P4：国家危机与精英分裂

财政、军队、合法性、粮运或继承至少两项严重失灵，统治联盟公开分裂。

- 主要玩法：紧急预算、军队忠诚、赈济、停战、组阁、借款、地方让步；
- 危机既可能打开改革窗口，也可能强化反动和军政动员；
- 出口：紧急妥协回 P2/P3；旧秩序强制恢复进 P5；双重权力进 P8；地方崩解进 P9。

### P5：反动、回撤与强制整合

受损集团借军队、宫廷、地产、信用、名分或外援回收权力，撤销或掏空改革。

- 主要玩法：清洗、复旧、收买、分化、舆论和强制；
- 可暂时恢复秩序，也会把矛盾压入地下；
- 出口：成功稳定到 P7 的保守版本；镇压失败回 P4/P8；经济衰退退到 P0 的弱化版本。

### P6：生产与组织跃迁

工业、交通、教育、资本和劳动组织跨过可持续阈值，旧制度无法只靠微调吸收变化。

- 主要玩法：决定所有制、劳动制度、市场法权、公共财政和政治代表；
- 不是“科技胜利”，而是新的阶级和国家能力迅速增长；
- 出口：新制度整合进 P7；分配冲突和政治排斥进入 P8；战争动员可能转 P4/P5。

### P7：新秩序整合

一种或多种新制度能够稳定征税、供军、分配、任命和处理冲突。

- 主要玩法：解决联盟内部矛盾、制定继承和反腐机制、将临时措施常规化；
- 可能是官僚工业国家、商业寡头、军事发展国家、大众参与政体或混合国家；
- 新秩序会产生自己的矛盾，数年后重新进入 P1。

### P8：双重权力与革命危机

旧国家与替代组织都能征收、动员、裁判或控制武装，主权不再唯一。

- 主要玩法：联盟扩张、军队倒向、地方接管、谈判、起义和制度纲领；
- 出口：革命/反革命胜利进入 P7；妥协宪制进入 P3/P7；失败回 P5；失控进 P9。

### P9：碎片化与战争经济

中央无法稳定控制税、军和交通，多中心政权争夺资源。

- 主要玩法：地区生存、联盟、统一战争、地方制度实验和外部干预；
- 工业和商业可能在局部继续，但全国市场和公共财政受损；
- 出口：新统一进入 P7；长期割据形成多中心常态；外来征服重置国家形式但保留部分社会结构。

## 4. 七个转型驱动值

局势不直接读取数百变量，而由年度调度器缓存七项 0—100 驱动值。

| 驱动 | 来源 | 高值意味着 |
|---|---|---|
| `productive_change` | 工业、商业、交通、城市与技术变化 | 物质结构快速改变 |
| `relations_mismatch` | 新生产与旧产权、劳动、税制不匹配 | 改革或冲突压力 |
| `state_crisis` | 国库、债务、军饷、粮运、合法性、继承 | 旧国家失去常规能力 |
| `elite_split` | 宫廷、官僚、士绅、军队、商人联盟分裂 | 政治机会打开 |
| `mass_mobilization` | C7/C8 组织、不满、传播和行动经验 | 民众能持续集体行动 |
| `alternative_capacity` | 地方议会、会馆、军政机构、改革官僚或革命组织 | 替代秩序可治理 |
| `external_pressure` | 战争、贸易封锁、外交、技术竞争、灾害 | 转型成本与紧迫性上升 |

## 5. 阶段进入与退出规则

### 5.1 组合判定，不用单阈值

示例：进入 P8 不能只要求民众不满 80，而应满足：

```text
state_crisis ≥ 65
elite_split ≥ 55
mass_mobilization ≥ 60
alternative_capacity ≥ 45
且存在至少一个替代联盟领袖或组织
```

若 `alternative_capacity` 很低，极端危机更可能进入 P9；若 `mass_mobilization` 很低而军队力量很高，更可能进入 P5 的军政整合。

### 5.2 滞后与最短阶段期

为防止阶段每年跳来跳去：

- 进入阈值高于退出阈值 10—15 点；
- 一般阶段最短持续 3 年，紧急危机可被重大事件提前打破；
- 同一转型方向失败后有 2—5 年政治疲劳；
- 阶段改变只在季度脉冲或重大事件结算；
- UI 提前显示“正在接近的阶段”与原因，不突然切换。

### 5.3 阶段不是万能效果包

阶段本身只提供规则参数、事件池、AI 优先级和少量修正。税收、军队、工业和阶级力量仍从底层制度计算。禁止给 P6 直接添加“发展 +50%、军队伤害 +50%”等无结构增益。

## 6. 多路线结果图

同一 P2/P3 改革竞争可产生不同长期制度：

| 路线 | 生产制度 | 国家形式 | 动员方式 | 主导联盟 | 主要代价 |
|---|---|---|---|---|---|
| 官僚整饬国家 | 商业农业/国家工业 | 官僚财政国家 | 税收—官僚 | 官僚整饬 + 皇权 | 官僚刚性、地方反弹、财政成本 |
| 宫廷工业集权 | 皇产/官营工业 | 宫廷家产国家 | 内廷财政 + 征发 | 宫廷集权 + 军事 | 公私混同、继承风险、商民排斥 |
| 商业法权秩序 | 私人工场和贸易资本 | 商业法权国家 | 市场—合同 | 商业技术 + 城市精英 | 寡头化、工资和城乡冲突 |
| 军事发展国家 | 军工和战略工业 | 军政动员国家 | 税官 + 军事征发 | 军事救国 + 技术官僚 | 民生挤压、军队掌权、永久战争 |
| 小农改良国家 | 分散地权、合作与公共工程 | 官僚/地方参与混合 | 税官 + 村社 | 民生平等 + 改革官僚 | 资本积累慢、地方差异、管理成本 |
| 大众工业共和国 | 公营、合作和受监管市场 | 大众参与国家 | 国民动员 + 自组织 | 劳动、小农、激进官僚 | 联盟碎片、供给与专业治理压力 |
| 士绅复旧 | 地租农业和有限商业 | 地方士绅共治 | 身份义务 + 地方中介 | 地方名分 | 国家税军弱、兼并和区域割裂 |
| 军阀资本主义 | 军镇工业、特许和掠夺贸易 | 多中心/军政 | 军事征发 + 合同 | 军头 + 军商 | 战争经济、劳动强制、统一市场破裂 |

路线不是互斥成就。国家可在不同区域呈现不同制度，并在一次危机后混合重组。

## 7. 改革与危机的节奏

建议一轮完整转型需要 15—40 游戏年，而不是数次点击：

```text
3—8 年：调查、试点、干部与联盟准备
3—10 年：法令竞争、预算和地方执行
5—15 年：基础设施、教育、工业与组织扩散
3—10 年：新利益冲突、危机或反动
5—15 年：新制度整合与继承考验
```

玩家可以通过战争胜利、没收和紧急权力加速，但速度越快，信息质量、财政、地方服从和联盟承诺越差，后续反弹越强。

## 8. 与原生王朝周期的关系

原生 `dynastic_cycle` 管理天命、帝国稳定与政治运动；新局势管理社会转型。桥接只使用少数催化剂：

| 新局势变化 | 对王朝周期 | 王朝周期对新局势 |
|---|---|---|
| 国家危机、欠饷、饥荒、内战 | 稳定下降 | 混乱提高 `state_crisis` 和 `elite_split` |
| 有效赈济、预算修复、军队国家化 | 稳定提高 | 稳定降低机会但提高长期改革执行 |
| 新制度合法化 | 可能恢复天命 | 天命只给正当性，不自动解决生产关系 |
| 革命/改朝换代 | 重置或改变王朝阶段 | 不清零土地、资本、组织和工业结构 |

**关键边界：** 改朝换代不等于社会革命；社会革命也不必立即改朝。一个新王朝可以继承旧土地关系，一个仍用旧国号的政权也可能彻底改变财政、军队和产权。

## 9. 玩家行动边界

玩家可以：

- 选择调查、试点、任命、预算和宣传；
- 与政治联盟交换职位、特权和政策承诺；
- 决定改革速度、补偿、镇压和地方差异；
- 建设工程、学校、军工和交通；
- 支持、分化或压制组织；
- 在危机中重组政府、借款、停战、赈济或军事动员；
- 在双重权力阶段选择妥协、革命、反革命或地方自治。

玩家不能：

- 直接给某阶级加减 30 点力量；
- 无条件宣布全国进入工业化或资本主义；
- 仅花威望让所有地方接受产权和劳动改革；
- 在没有干部、预算、交通和组织时建立全国新制度；
- 通过一次处决永久消灭一个物质利益集团。

## 10. AI 路线选择

AI 不预先绑定“历史路线”，而根据统治联盟、危机和能力评估方案：

```text
route_score = 生存收益
            + 统治联盟物质收益
            + 领袖思想匹配
            + 可执行性
            + 外部威胁适配
            - 财政成本
            - 联盟分裂风险
            - 革命/政变风险
```

AI 具有路线惯性，除非危机或领袖更替，不每年改主意。失败会留下“政策记忆”：同一联盟对曾导致灾难的手段降低偏好，对成功组织和工程提高信心。

## 11. 性能与数据规则

- 七个驱动值每年重算一次，重大事件只加即时偏移；
- 阶段转换按季度检查，只检查当前阶段允许的出口；
- situation 成员每季度重算，限定重要人物和相关政权；
- 阶段历史只保存最近 10 次转型和关键原因；
- 每项改革/革命最多一个主任务链，其他反应用聚合事件；
- 不为每个区域运行独立完整 situation；区域只保存结构轴和执行状态；
- 阶段 UI 读取缓存，不动态全图统计。

## 12. 验收情景

- [ ] 高商业化但土地集中、工资劳动低时，不会自动进入资本工业阶段；
- [ ] 工业能力上升但国家危机低、制度能吸收时，可长期停留在改良秩序；
- [ ] 民众不满极高但替代能力低时，更可能碎片化而非自动革命；
- [ ] 军队强、国家危机高、群众组织低时，优先出现军政整合；
- [ ] 改革法令通过后可出现地方妥协、形式执行和反向转嫁；
- [ ] 新秩序整合后仍会产生新的结构张力，不进入永久黄金时代；
- [ ] 改朝换代不会清零生产关系，社会革命也不强制更换国号；
- [ ] 同一工业能力可进入官僚、宫廷、商业、军事或大众路线；
- [ ] 阶段有进入/退出滞后、最短持续时间和可解释原因；
- [ ] 原生王朝周期与新局势互相桥接但不相互覆盖。

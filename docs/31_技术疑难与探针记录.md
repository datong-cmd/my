## 已确认的低风险事实

- 《变身大明》以 `is_GM_emperor = yes` 判定 `has_title = title:h_greatming`；伴生模组只读该 trigger。
- 原模组已在独立文件追加 `on_title_gain`、`yearly_global_pulse`，说明这两个 on_action 允许合并追加。
- 本体标注 `yearly_global_pulse` 为无 root 的年度入口；本模组从 `h_greatming` 单点进入，不扫描人物或地图。
- 原模组已经把变量直接存放在 `title:h_greatming`，因此伴生模组采用同一存储层、但全部使用 `minghm_` 前缀隔离。
- `tools/verify_reference_interfaces.ps1` 会在每次实现前重新核验上述接口以及 `HQ_change` 的双校验常数、`-500..2000` 钳制、朋党数值与本体 pulse；任一接口变动即中止对应桥接的静态通过。

# 《变身大明》源码文件索引

> 生成范围：`common/`、`events/`、`gui/`、`history/`、`localization/`、`map_data/` 下的文本逻辑文件。美术、音乐与字体只做资产分类，不逐文件解释。
>
> 定位格式统一为 `相对路径:行号`。‘覆盖本体’只表示游戏本体存在完全相同的相对路径，意味着兼容性风险高，不等于已判定为完整覆盖。

## 使用方法

- 想找某个机制：先查后续的“机制→源码”人工索引，再回到本文件确认同目录相关实现。
- 想找某个脚本 ID：查 `02_脚本ID定位索引.md`，其中保留顶层项的起始行号。
- 想判断是否借用了本体文件：看“覆盖本体”列；标为“是”的文件必须做逐段差异分析。
- `.txty/.txtz/.guiz` 均为可读文本，只是改了扩展名；索引仍将其视作脚本。

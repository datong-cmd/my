# 30C 事件链三：边战失利与军制重整

## 1. 入口

主要入口为明帝相关战争结束且失败、白和但损失严重，或防御战争持续两年以上。实际战争事实来自本体 on_action；不扫描所有战争。和平时期只有 `military_autonomy ≥70` 且危机压力高时才可由年度事件触发“军令不一”变体。

初值：

```text
support = state_capacity/2 + 原军职/军功人物支持 + 皇权临时修正
resistance = military_autonomy + crisis_pressure/2 + 将领利益修正
urgency = 战争进行中 90；战后 60
```

## 3. 本体/DLC边界

- 不生成敌国、战争或军队来配合剧情；有真实战争就读取，没战争就用事件演训。
- 不新增 MAA，不改参考模组军功、卫所、五军都督府和兵种定义。
- 将领使用现有五军职位、军职、军功人物；无合适人物时事件以“诸将联名”匿名表示。
- 厂卫核查只处理本链支持/阻力，不外部触发 `GM_Tanhe_events`。

## 4. 结局对价

| 结局 | 收益 | 代价 |
|---|---|---|
| 文官控制 | 国家能力 `+6`、军事自主 `-6` | 战时响应短期下降、守成力量 `+3` |
| 军政合作 | 国家能力/精英协调各 `+4` | 军事自主 `+2`，需兑现军职承诺 |
| 技术军备转向 | 开启 `30D` 候选、改革势头 `+5` | 商业影响或军事自主 `+4` |
| 军头化 | 当前危机响应高 | 军事自主 `+10`、双重权力风险上升 |
| 整顿失败 | 无结构收益 | 危机压力 `+8`、改革势头 `-5` |

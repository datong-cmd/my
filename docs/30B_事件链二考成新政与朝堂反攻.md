# 30B 事件链二：考成新政与朝堂反攻

## 1. 入口

玩家决议发起，或 AI 在 `reform_momentum ≥45`、`state_capacity ≤65`、无主链时年度加权选择。已有继承链须完成后才能启动。它是 `23A/23B` 的首个完整政策垂直切片。

初值：

```text
support = reform_momentum + state_capacity/2 + 主持官能力修正
resistance = conservative_power + bureau_capture临时快照/2 + 党派反对
urgency = 40 + crisis_pressure/3
commitment = 玩家选择的改革深度 30/60/90
```

## 3. 核心数字

每个执行节点：

`本轮落实 = support + 主持官适配 + 信息质量 - resistance - 改革深度/3 + 0..10`

成功不会只加国家能力：主持官及其朋党会获得政治资本；频繁撤换增加轮换成本；厂卫解决阻挠会增加恐惧/信息失真（落实时体现为额外阻力）。

## 4. 结局

- 制度化：国家能力 `+8`、改革势头 `+6`、守成力量可能 `+2`（官僚掌握新制度）。
- 妥协化：国家能力 `+4`、精英协调 `+4`、改革势头 `-2`。
- 纸面化：危机压力 `+5`、守成力量 `+5`，主持人可被追责。
- 反噬：改革势头 `-8`、守成力量 `+8`，候选触发 `30E` 大案或党争。

所有幅度为初值，总量钳制。不得修改原品级、科举结果或正式官位。

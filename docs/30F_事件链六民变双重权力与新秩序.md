# 事件链六：民变、双重权力与新秩序

> 表现旧统治不能照旧运行、被统治者不愿照旧生活时的力量重组；重点是长期博弈与可逆阶段，而非一次按钮推翻王朝。

## 1. 触发与阶段边界

- 通常要求历史阶段已到“结构危机”或以后；民众动员、危机压力较高，且国家能力或精英凝聚明显不足；
- 可由前五条链失败留下候选标记；全国无其他重大链；预期 2—6 年；
- 仍只处理 `h_greatming`，不创建新国家、市场、阶级人口或全国地图参与者。

“双重权力”是朝廷命令与地方/社会动员谁能执行的抽象状态，不生成第二套头衔政府。

## 2. 力量对比

```text
变革力量 = 民众动员 + 改革动量/2 + 危机压力/2 + 路线加成
王朝力量 = 国家能力 + 精英凝聚 + 皇权临时修正 + 军队忠诚
```

- 差值钳制在 -200—200，只在固定节点重算；
- 路线：1 恢复秩序、2 自上改革、3 宪制妥协、4 革命重组、5 军事反动；
- 群体以匿名文本表现；优先选择现有朝臣、军职和党争人物；
- 仅在进入真实战争且无有效领袖时，最多生成一名长期行动者。

## 4. 五种终局

- 王朝明显占优 + 恢复秩序：短期能力上升，未兑现诉求抬高后续危机；
- 王朝略占优 + 自上改革：保留明制，改革动量和制度承诺上升；
- 双方接近 + 宪制妥协：皇权下降，精英凝聚与国家能力重组；
- 变革明显占优 + 革命重组：进入“新秩序”，动员转化为制度能力，但保守力量不清零；
- 军队决定胜负：军事反动，国家能力暂稳、军事自主显著上升。

单项结算通常不超过 20，历史阶段最多变化一级，任何结局都不把八项全国值拉满或清零。

## 5. 战争桥接技术闸门

首版默认事件数字对决。真实战争只在以下条件全满足时开放：

1. 已验证可安全调用的既有 CB 与战争结束作用域；
2. 不覆盖中国皇权、天命、朝贡或《变身大明》职位；
3. 失败时可回落到 `.6070` 数字判定；
4. 全国仍只有一条重大链；
5. 不批量创建军队、人物或地图参与者。

因此“革命”首先是统治关系变化，战争只是可选表现层。

## 6. 本体边界、AI与验收

- 只用 story、书信/角色/隐藏事件、角色修正、头衔变量与延迟事件；
- 复用暴动、御前、军营、城镇、诏书背景和图标，不新增 3D；
- AI 根据力量差、性格和路线选择，不搜索全国角色或领地；
- 必测五终局、继承、关键人物死亡、无战争回落、探针关闭、连续危机冷却；
- 结局写回历史阶段、承诺和八项全国值，并清理全部临时 scope、story 与重大链锁。

# 33 2D 图标资产清单

## 1. 用途与边界

本清单记录伴生扩展唯一新增的 2D 资源。它们是 64×64、带透明通道的 DDS 行内图标，注册于 `mod/minghm_companion/gui/minghm_texticons.gui`，仅由本地化中的 `@minghm_*!` 调用。

- 不创建独立窗口、轮盘、动画、modifier/trait 图标槽或运行时更新；
- 不复制本体、DLC 或《变身大明》的任何二进制资产；
- 不增加脚本脉冲、人物、地区或经济计算；
- 运行时只多读取 6 个合计不足 100 KiB 的静态纹理。

## 3. 来源与处理

六枚图标均按本项目的抽象机制描述生成，未使用上游角色、模型、场景或原始贴图。发布版在去除临时底色后缩放为 64×64 RGBA DDS；本地可编辑源图和检视图不随 Git 发布，以免无谓增大仓库。

## 4. 晚间实机检查

1. 同时启动 CK3、本体 DLC、《变身大明》和本模组；
2. 打开“查看大明局势”，确认五枚阶段图例均正常显示、没有紫色底或缺失纹理；
3. 用控制台或正常流程触发 `minghm_core.0005`，确认当前阶段选项前只有对应图标；
4. 检查技术组织和新秩序相关决议、事件标题是否显示图标；
5. 若日志出现 `texticon` 或 DDS 读取错误，先保留截图和 `error.log`，再回退为纯文字显示。

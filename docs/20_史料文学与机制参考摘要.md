# 史料、文学与机制参考摘要

> 目的：只保留能直接服务机制设计的参考框架。这里不是明史综述，也不要求事件逐年复刻。
>
> 使用原则：史实提供矛盾结构、行动语言和约束；游戏机制决定节奏、可玩性和多结局。

## 1. 证据如何使用

### 1.1 五级用途

| 类型 | 可以支持 | 不可以直接支持 |
|---|---|---|
| 制度原始材料 | 机构名义职责、程序、奏议语言和政策主张 | 机构实际总按条文运行 |
| 财政/军事账册 | 收支、军费、运输和征收矛盾的量级与类别 | 未记录部分不存在、账面等于实际 |
| 实录、传记、文集 | 精英行动、联盟、舆论和事件顺序 | 全社会利益、作者没有立场偏差 |
| 地方志、契约、行业材料 | 区域差异、土地/劳动/市场关系 | 全国统一模式 |
| 现代研究与文学 | 分析问题、比较模型、情景灵感 | 作为单一无争议事实或逐句素材 |

### 1.2 机制证据标记

- `H1`：多个材料支持的结构关系，可进入核心机制；
- `H2`：特定时期/区域的强案例，适合事件族或开局参数；
- `H3`：有争议解释，只能作为一条路线/叙事立场；
- `L`：文学启发，只借鉴博弈结构；
- `A`：为玩法做的抽象，不声称史实存在同名制度。

实现文档原有 `S1/S2/S3/D/R` 用于脚本证据；本表 `H/L/A` 用于历史与叙事证据，两者不能混用。

## 2. 最值得保留的六条结构认识

### 2.1 国家不是单一行动者

皇帝、内廷、内阁、部院、言官、地方官、士绅、军镇和税役中介掌握不同信息与资源。中央颁令必须经过人事、账册、地方协作和强制才能变成结果。

机制转化：

- 国家能力拆成财政、官僚、信息、强制、基础设施和合法性；
- 皇权控制与公共国家能力分开；
- 政策有提议、结盟、试点、扩散、反制和结算；
- 地方执行有折损与负担转嫁；
- “强中央”可能是规则化国家，也可能是宫廷私产化或军政强制。

### 2.2 固定制度会被货币化、战争和区域分化改变

税额、身份役、军户、宗藩供养和低官俸在商业化、银流、军费和地方市场变化下出现错配。晚明军费、粮食市场、边防银流、权贵土地与商业机会彼此塑造，而不是“财政”和“市场”两个孤立系统。[军事制度研究](https://www.cambridge.org/core/journals/journal-of-chinese-history/article/why-military-institutions-matter-for-ming-history/B3C345236C8588E708C661758D2DED51)、[《万历会计录》与财政主权研究](https://hub.hku.hk/handle/10722/309698)

机制转化：

- 一条鞭/货币化降低部分行政复杂度，同时增加现金和价格风险；
- 战争会制造商人、军需、运输和军头利益，也会加重摊派；
- 国库数字不等于征收能力，信用与税基可见度单列；
- 制度收益与制度适配分开，旧制度可在部分区域继续有效。

### 2.3 “士绅、商人、农民”都必须拆开

官僚服务、地主产权、功名网络、商人信用、工场生产、小农家庭、佃户和雇工处在不同生产关系中。匠籍义务、有偿劳动、家庭作坊、包买制和集中雇工可以并存；商业组织也可能同时协助国家、维护准入和形成自主集体行动。[劳动与商业化研究](https://www.cambridge.org/core/journals/international-review-of-social-history/article/work-ethics-and-work-valuations-in-a-period-of-commercialization-ming-china-15001644/4FA7B22851A0B0D2E37700E494A6570B)、[行业组织综述](https://www.cambridge.org/core/journals/international-review-of-social-history/article/chinese-guilds-from-the-seventeenth-to-the-twentieth-centuries-an-overview/61A96EB7FF67CE0BE8073C36DDB049CD)

机制转化：

- C2 官僚—士绅地主与 C3 国家官僚服务层分开；
- C5 商人金融与 C6 工场/技术生产者分开；
- C7 自耕编户与 C8 佃户/雇工/依附劳动分开；
- 阶级以物质基础、组织、国家入口、强制、合法性、凝聚和不满表达；
- 人物身份只是代理信号，不直接等于阶级立场。

### 2.4 商业化和资本形成不是同一件事

贸易利润可以流向地产、消费、借贷、官位、特许或生产；集中生产也可能由宫廷、国家、军镇、商人、作坊联合或劳动合作掌握。“资本主义萌芽”本身是有争议的史学解释，不宜作为必然阶段。[相关史学讨论](https://journals.sagepub.com/doi/10.1177/009770048200800103)

机制转化：

- 商业化、工资劳动和工业能力是三个区域轴；
- 工业化需要资本、原料/能源、技术、劳动、运输、需求和再投资闭环；
- 所有制决定收益、控制和反制；
- 资本可能生产化、地产化、金融化、官僚化、军事化或外逃；
- 不预设资本主义、革命或官僚国家必胜。

### 2.5 群众不满不等于革命

灾害、赋役、地租、债务、欠饷和镇压会产生不满，但持续政治行动需要组织、共同经验、传播、领导、精英裂缝和替代治理能力。

机制转化：

- grievance 与 power/mobilization 分离；
- 请愿、逃亡、抗租、罢市、兵变、地方自治、双重权力形成动员阶梯；
- 革命进入条件组合 `国家危机 + 精英分裂 + 群众组织 + 替代能力`；
- 镇压能降低公开行动，却可能保留仇恨、组织记忆和生产破坏；
- 革命胜利之后仍需财政、粮食、军队、官僚和合法性整合。

### 2.6 边疆是国家形成的一部分

土司、军镇、贡属、移民、贸易、军粮和征服共同体不是“地图边缘事件”。中央通过不同契约取得税、兵、礼仪与战略空间，地方则保存土地、继承、武装与自治。

机制转化：

- 贡属关系拆成礼仪、贡赐、贸易、军事义务和干预；
- 归流可逆且会损失地方信息；
- 殖民开发同时改变土地、劳动、财富输送和族群政治；
- 征服共同体有供养、从征、身份分配和改革矛盾；
- 中心危机提高外围自治和替代中心能力。

## 3. 明代事件母题到游戏机制

这些母题不要求固定年份和历史人物。每项都可由满足结构条件的架空人物触发。

| 历史母题 | 核心矛盾 | 机制入口 | 不应写成 |
|---|---|---|---|
| 土木危机式军事失败 | 皇权亲征、军队指挥、继承与首都防卫 | 战败—继承—拥立—军政联盟危机 | 固定人物必败事件 |
| 夺门/拥立式政变 | 合法性、军功、宫廷与官僚承认 | 精英分裂、联盟承诺、短期强制 | 单次 intrigue 骰子 |
| 大礼议式礼制冲突 | 皇统名分与官僚解释权 | 意识形态霸权、官职清洗、路线记忆 | 纯 opinion 争吵 |
| 倭患与海禁 | 海贸、走私、沿海防务和特许 | 开放/封锁/许可贸易、地方武装、商人联盟 | “开放必好/封禁必坏” |
| 北虏与边防 | 军费、马市、军镇和战略纵深 | 贡市—战争—军粮—军头政治 | 只加外敌兵力 |
| 张居正式整饬 | 考成、清丈、财政和皇权授权 | 多年改革、官僚执行、士绅反制 | 一键改革 buff |
| 万历式国本/怠政 | 继承、任命否决、财政拨付和官僚抵制 | 长期宪制危机、职位空缺、消极否决 | “懒惰皇帝”性格惩罚 |
| 矿税/榷税 | 财政紧急、内廷代理、商人和地方社会 | 非常征收、抄核、罢市、国家私产化 | 单纯加税率 |
| 东林—阉党式冲突 | 书院/言官网络、内廷情报、任命与案件 | 联盟组织、舆论、监察、清洗、反清洗 | 善恶二元阵营 |
| 辽东军费与军镇 | 欠饷、指挥、军功、军器和地方供给 | 财政—军事复合危机 | 单纯战争 debuff |
| 灾荒、流民与民变 | 生计、运输、征收、债务和组织 | 救济、减免、迁徙、抗租、招抚、镇压 | 灾害后自动叛乱 |
| 银流/信用冲击 | 货币税、价格、贸易和债务 | 区域差异、信用危机、税制再议 | 单因论式“白银导致灭亡” |
| 王朝崩解 | 财政、军队、精英分裂、民众组织与外压耦合 | P4/P8/P9、多中心秩序 | 固定日期 game over |

## 4. 历史阶段的使用方式

历史唯物主义在本 Mod 中不是“封建→资本主义→革命”的关卡表，而是：

```text
生产力变化
↔ 土地、劳动、产权和分配关系
↔ 国家征收、信息和强制形式
↔ 阶级组织、联盟和意识形态
↔ 战争、市场、灾害和国际压力
```

阶段 P0—P9 是上述关系的暂时结构，不是年份。相同技术水平下可以出现宫廷家产国家、官僚财政国家、地方士绅共治、商业寡头、军政动员或大众参与等不同组合。

## 5. 《万历明君》的有限参考

用户所指文学参考采用鹤招《万历明君》。其玩法价值不是让玩家复制主人公答案，而是提供以下 `L` 级结构灵感：

- 弱小统治者必须在首辅、次辅、内廷、太后、言官与地方网络间借力；
- 任命、仪礼、财政和私人关系都可成为政治交换；
- 改革同盟不稳定，今日协作者可能在执行范围和利益分配上成为反对者；
- 中央意图进入地方后会经历外包、虚报、刚性执行、地域差异和负担转嫁；
- “知道正确政策”不等于有资源、组织和合法性执行；
- 国家弱小时的外交、军备和财政决策互相制约。

公开可访问的作者结卷说明也强调弱小时的纵横捭阖、国家危弱时的扶危与清丈执行变形，这与本策划的联盟和区域执行模型一致。[作者结卷说明镜像](https://www.mingzw.net/mzwread/42600_13013038.html)

边界：

- 不复制小说段落、对白、原创人物弧和具体桥段；
- 不把小说解释当史实；
- 不把幼帝或特定人物设为玩法必要条件；
- 只转化为通用的弱权力、结盟、信息和执行机制；
- 出版/授权不明确的文本不打包进 Mod 或仓库。

## 6. 后续需要按需查的史料类型

编码阶段只有具体机制需要时才查：

| 实现任务 | 优先材料 | 要提取的字段 |
|---|---|---|
| 官职与程序 | 会典、实录、官箴、奏疏 | 名义职责、任命、任期、相互否决 |
| 财政与预算 | 会计录、赋役材料、奏疏 | 收入类别、支出顺序、拖欠、地方留成 |
| 土地与清丈 | 鱼鳞册/黄册研究、地方志、契约 | 登记、隐田、执行者、转嫁、争讼 |
| 军户与募兵 | 兵制、军饷、卫所和军镇研究 | 身份、土地、军功、欠饷、私人依附 |
| 商业与工场 | 契约、会馆/公所、行业和城市研究 | 信用、订单、劳动、准入、集体行动 |
| 灾荒与民变 | 地方志、赈灾奏疏、案件材料 | 粮价、迁徙、救济、组织、镇压与招抚 |
| 土司/边疆 | 请封、盟约、地方志、边政材料 | 继承、贡赋、兵役、土地、归流和反抗 |
| 意识形态 | 奏议、书院文集、告示 | 谁以何种名分为哪种利益辩护 |

每次只回答当前机制的 3—5 个问题，不做无边界资料堆积。

## 7. 史料转机制卡片

新增历史内容先填：

```text
材料/出处：
时段与区域：
证据级别：H1/H2/H3/L/A
行动者：
可控制资源：
制度入口与否决点：
信息不对称：
物质受益者/承担者：
组织与强制：
意识形态语言：
短期结果：
长期结构变化：
可替代人物/条件：
映射的现有变量和公共接口：
为什么值得成为选择而非背景文本：
```

如果无法映射现有状态，先判断是资料只适合叙事，还是数据字典确实缺一个普遍机制。不能为一条轶事新增永久变量。

## 8. 事件文本的历史感

历史感主要来自：

- 人物以名分、祖制、民生、军国、财用、地方情势和职责说话；
- 同一政策由不同利益集团给出不同合理解释；
- 信息通过奏疏、账册、揭帖、审讯、商路、军报和流言不均匀传播；
- 成本具体落在俸、饷、粮、地、役、债、工、税、职位和时间；
- 地方和中央对“成功”的定义不同；
- 旧制度不是愚昧残余，而是曾解决过成本和协调问题的安排。

不需要堆叠冷僻官名、年号和长篇史料引用。玩家首先要看懂谁要什么、凭什么、谁付钱、谁能阻止。

## 9. 需要避免的史观与玩法误区

- 避免把皇帝个人勤惰当作全部解释；
- 避免把士绅、宦官、商人、军人或农民写成同质善恶阵营；
- 避免以“腐败”覆盖低俸、预算缺口、地方中介和制度冲突；
- 避免把商业化直接等于资本主义或公共富裕；
- 避免把技术发明直接等于可持续生产能力；
- 避免把灾害直接等于王朝崩溃；
- 避免把镇压写成无成本稳定，也避免把起义写成自动进步；
- 避免固定结局和主角知识造成一边倒；
- 避免用现代政治术语替代人物时代语言，但系统层可以清楚解释结构；
- 避免为了真实性做逐人逐户模拟，破坏性能和可玩性。

## 10. 参考使用结论

这批材料已经足以支持第一轮机制实现。后续史料工作应由具体需求触发：例如 M2 做重大案件时再核对审理程序，M3 做清丈时再核对执行主体和地区差异，M4 做欠饷时再核对军镇资源链。

在没有新实现问题时，继续扩充史料的收益低于先完成 M0/M1 技术探针与垂直切片。

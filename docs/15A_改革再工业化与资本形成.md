# 改革、再工业化与资本形成

> 文档性质：改革项目模板、工业化资源闭环与资本主义多路径设计  
> 原则：改革改变规则和力量分配；工业化需要完整再生产链；资本主义是一组生产关系，不是奖励标签。

## 1. 改革的统一数据结构

每项改革都必须先写一份“改革案”，不能直接从创意跳到事件代码。

| 字段 | 必填内容 |
|---|---|
| 问题定义 | 哪个结构矛盾使旧制度失效；谁认为它是问题 |
| 目标规则 | 要改变产权、税制、劳动、任命、军队、市场还是政治代表 |
| 发起联盟 | 皇帝、部院、地方官、商会、军队或民众组织 |
| 执行载体 | 法律、契约、官职、工程、活动、互动或战争 |
| 受益者 | 获得收入、权利、职位、组织或安全的集团 |
| 承担者 | 失去收入、特权、土地、控制或预算的集团 |
| 资源需求 | 国库、人员、信息、军队、交通、技术和时间 |
| 地方适配 | 哪些区域先试、哪些区域阻力高、是否允许不同版本 |
| 失败形态 | 拖延、形式执行、转嫁、腐败、反抗、财政中止或政变 |
| 维持条件 | 新制度如何持续筹资、培训、监督、继承和纠错 |
| 退出/回滚 | 何时暂停、修改、废止；沉没成本如何处理 |
| 可解释反馈 | 玩家如何看见进度、赢家、输家和失败原因 |

任何缺少“承担者、失败形态和维持条件”的改革案不得进入编码。

## 2. 改革生命周期

```text
发现问题
  ↓
调查与形成知识
  ↓
构造发起联盟
  ↓
争取法令与预算
  ↓
选择试点和执行者
  ↓
地方贯彻 / 妥协 / 形式执行 / 转嫁 / 抗拒
  ↓
审计、补偿、镇压或修订
  ↓
新制度再生产或回撤
```

### 2.1 六项执行能力

| 能力 | 载体 | 缺失时的失败 |
|---|---|---|
| 法律授权 | 皇权、议政、部院、合法性 | 地方称法令无效或越权 |
| 财政保障 | 国库、固定预算、信用 | 拖欠、停工、卖权和摊派 |
| 人员能力 | 功绩、官员、技术人员和基层吏员 | 报表造假、简单粗暴、依赖旧中介 |
| 信息能力 | 清丈、统计、巡察、市场与情报 | 目标错误、隐匿、虚增和政策滞后 |
| 强制后盾 | 司法、监察、军队和治安 | 有组织既得利益公开抗拒 |
| 社会合作 | 小业主、商人、工匠、村社和工人组织 | 成本转嫁、消极执行、地下市场 |

改革者不能只提高强制力。强制能突破首次抗拒，却不能替代长期信息、财政和合作。

## 3. 八条改革路线

### R1：财政—官僚整饬

核心：清丈、统一册籍、考成、预算、增俸、审计和地方留成规则。

- 发起者：官僚整饬联盟、中央皇权；
- 受益：职业官僚、守法小业主、需要稳定合同的商人；
- 受损：隐田士绅、依赖模糊账目的中介、内外财政割据者；
- 成功：国家可见税基和稳定预算上升；
- 变形：指标主义、虚报增亩、负担转嫁、官僚垄断信息；
- 长期矛盾：国家能力提高后，是用于公共品还是扩大征取。

### R2：宫廷集权与内廷直达

核心：以内官、厂卫、皇产、直接任命和特使绕开常规官僚。

- 优势：危机时速度快，能打破地方保护；
- 受益：皇室宫廷、依附内廷的新官僚和皇商；
- 受损：部院、地方士绅、公共预算；
- 变形：公私财政混同、任意征收、继承时代理网络崩解；
- 可发展为宫廷工业集权，也可能退化为掠夺性皇产经济。

### R3：地方士绅修复

核心：减轻中央指标、依靠乡约宗族、保护地方产权和调解权。

- 优势：治理成本低、地方知识强、短期恢复秩序；
- 受益：士绅地主、小业主、地方官；
- 受损：中央税军、流动人口、被地方权势排斥者；
- 变形：隐田、地方俘获、宗族压迫和区域割裂；
- 不是纯反动：可提供教育、水利和救济，但公共性取决于监督和竞争。

### R4：商业法权与市场统一

核心：统一关卡、货币和度量衡，稳定合同、商税、信用、破产和合股规则，开放海贸和跨区市场。

- 受益：商人、作坊主、运输和城市消费者；
- 受损：地方税卡、特许中介、走私保护者；
- 变形：商人寡头、特许垄断、债权压迫、区域产业被竞争摧毁；
- 长期选择：利润流向土地/官位，还是流向生产设备、技术和工资劳动。

### R5：国家主导再工业化

核心：国家军器局、矿冶、交通、技术学校、标准化、采购和战略工业。

- 受益：技术官僚、军队、国营工场、工匠和部分劳动者；
- 受损：旧工匠役制中介、私人垄断者、被征用土地和劳役者；
- 优势：能集中承担大项目和长期风险；
- 变形：官营低效、腐败承包、预算挤压、军工优先和军头控制；
- 成功条件：公共预算、审计、技术晋升、稳定需求和民用扩散。

### R6：私人资本与工场扩张

核心：让商业资本进入生产，支持合股、信用、设备、雇佣劳动、城市和出口市场。

- 受益：商人资本、工场主、技术人员，初期也增加雇工机会；
- 受损：传统作坊、受竞争冲击地区、被圈占土地者；
- 优势：响应市场快，国家直接成本低；
- 变形：垄断、低工资、债务失地、资本外逃和政策俘获；
- 成功条件：利润再投资、劳动力技能、市场规模、产权和基础设施。

### R7：小农、土地与劳动改良

核心：减租、赎买、限田、债务调整、村社公共土地、劳动保护、救济和基层参与。

- 受益：小农、佃户、雇工、合作生产；
- 受损：大地主、高利贷和低工资雇主；
- 优势：提高生计、内需和政治参与；
- 变形：假分户、资本转移、地主武装反抗、生产组织碎片化；
- 成功条件：财政补偿、土地登记、农业公共品、信用替代和市场接入。

### R8：军事—动员现代化

核心：足饷、募兵/征兵、军官教育、军工、后勤、军队国家化和战略交通。

- 受益：国家军队、军工、技术官僚、军需商人；
- 受损：世袭空额、私人家丁、地方军事垄断；
- 优势：外压下迅速提升生存能力；
- 变形：军政专断、预算吞噬、军头资本和永久紧急状态；
- 成功条件：文官预算控制、军官轮调、可靠饷源和退役安置。

## 4. 再工业化的七环闭环

工业能力不能靠一个创新或建筑持续。每个产业链至少需要：

```text
农业与生计剩余
  → 可投资资本与财政
  → 原料和能源
  → 技术、教育与标准
  → 稳定劳动与管理
  → 交通、市场和需求
  → 设备维护与利润再投资
  ↺
```

### 4.1 七环指标

| 环节 | 主要变量 | 断裂后果 |
|---|---|---|
| 生计剩余 | 粮食安全、小农收入、城市供给 | 工资上涨、饥荒、工人离散、征粮冲突 |
| 资本/财政 | 国库、信用、商人资本、储蓄 | 项目停工、设备不足、卖权或外债 |
| 原料/能源 | 铁、煤、木、硫硝、棉麻、水力 | 成本上升、产业受制于单一区域 |
| 技术/标准 | 技术教育、工匠网络、度量衡、图纸 | 样品成功但无法复制，质量不稳 |
| 劳动/管理 | 雇工、技师、经理、纪律与权利 | 缺工、逃亡、破坏、低效率或冲突 |
| 交通/需求 | 运河、道路、港口、军队和民用市场 | 库存积压、区域孤岛、军工单一依赖 |
| 维护/再投资 | 利润留存、折旧、维修、零部件 | 一代工程后退化，技术人员流失 |

工业化项目完成只提高某几环，不能直接宣布完成再工业化。

## 5. 五类国家能力指数

为避免逐件商品，国家层只保存五项聚合能力：

| 能力 | 表示 | CK3 来源 |
|---|---|---|
| 民用生产能力 | 纺织、纸、陶瓷、工具、消费品与设备供给 | 区域工业轴、城市、工场和项目 |
| 军事工业能力 | 火药、兵器、火炮、船舶和维修 | 火药库、军器局、矿冶、军费 |
| 运输物流能力 | 粮运、道路、运河、港口、仓储和组织 | 大型工程、发展、控制与合同 |
| 技术扩散能力 | 教育、印刷、标准、师徒、研究和复制 | 家产建筑、活动、学校和人才 |
| 能源材料能力 | 矿冶、燃料、木材、化学原料与水力 | 资源区、建筑、工程和贸易 |

每项 0—100，全国由区域加权。兵种、工程和阶段要求不同组合；不新增钢、煤、火药等逐月库存，除非重大危机需要临时短缺状态。

## 6. 工业项目分级

### I0：制度准备

清丈、度量衡、预算、合同、技术调查、工匠登记改革和学校。主要改变信息、标准和干部，不立即给大量产能。

### I1：区域样板

改良矿场、纺织作坊联合、造船所、纸坊、火药作坊或水利。验证技术和组织，失败成本可控。

### I2：国家工程

运河整治、国家军器局、大型矿冶、港口、海塘、交通干线。使用原生 great project，允许多方出资和政治交换。

### I3：产业网络

把原料区、生产区、运输、市场、维修和教育连接起来。此时才显著提高持续产能。

### I4：机器与能源跃迁

蒸汽/机械、标准零件、大型动力和新交通。需要前四级积累，不能由穿越知识直接跳过；失败会留下昂贵样机和债务，而非无效果。

## 7. 技术知识的边界

“知道原理”只提供研究方向或降低首次试验成本。实际采用还需要：

- 能制造所需材料和精度；
- 能训练技术人员和管理者；
- 有稳定动力、维修和零部件；
- 有买得起产品的国家或市场需求；
- 有能承担失败的财政与组织；
- 有政治联盟保护项目不被既得利益中止。

穿越者或天才人物死亡后，只有写成标准、学校、组织和生产流程的知识才能保留。人物加成不能成为唯一事实源。

## 8. 工业化融资与权力后果

| 融资方式 | 立即效果 | 获得权力者 | 长期风险 |
|---|---|---|---|
| 常规国库预算 | 稳定、公共性高 | 财政官僚和议政机构 | 挤压军费/赈济，税负争论 |
| 特别加派 | 快速筹款 | 中央与征收者 | 小农破产、不满、逃税和民变 |
| 内帑/皇产 | 绕开部院 | 宫廷、宦官、皇商 | 公私混同、继承和监督风险 |
| 商人垫款/国债雏形 | 迅速获得银与信用 | 金融商人、债权人 | 债务主权、利息和政策俘获 |
| 特许垄断 | 国家现金成本低 | 特许商人和官商集团 | 高价、寻租、竞争受阻 |
| 没收/抄家 | 一次性巨额资源 | 执行机构和受分配者 | 产权恐惧、资本外逃、政治清洗 |
| 海贸关税 | 随贸易增长 | 港口、海商、海关官僚 | 外部封锁、走私和沿海偏重 |
| 地方合资 | 共享成本和利益 | 地方官、士绅、商会 | 区域自主、收益截留 |
| 劳动/社区出资 | 低现金、高参与 | 村社、工匠和劳动组织 | 强制摊派或协调困难 |

玩家选择融资方式就是选择未来谈判对象。钱不是中性资源。

## 9. 资本的四种形态与转化

| 资本形态 | 收益来源 | 优先投资 | 政治偏好 |
|---|---|---|---|
| 地租资本 | 土地租金、债务取得土地 | 地产、水利、功名 | 地权、低税、地方秩序 |
| 商业金融资本 | 贸易差价、信用、税契与特许 | 商路、库存、借贷、官位 | 市场准入、货币稳定、特许或法权 |
| 工业生产资本 | 设备、雇工、技术和规模利润 | 工场、机器、培训、原料 | 产权、劳动纪律、市场和基础设施 |
| 国家/军事资本 | 公共税收、军需和战略投资 | 军工、交通、矿冶和教育 | 统一、预算、动员和国家控制 |

资本会在四者间流动。工业利润低或产权不稳时，商人买地、放贷、捐纳和购特许；国家工程可靠、市场大、劳动力和技术充足时，商业资本才更愿意进入生产。

### 9.1 资本转化评分

```text
industrial_investment_attractiveness
  = expected_market
  + infrastructure
  + technology
  + contract_security
  + labor_supply
  + state_orders
  - confiscation_risk
  - war_risk
  - input_cost
  - landed_return
  - monopoly_entry_barrier
```

玩家不能命令私人资本“投资工业”，只能改变收益、风险、组织和政治承诺。

## 10. 资本主义形成判定

资本主义不是 `has_flag = capitalism`。建议从五项关系判断：

| 条件 | 说明 |
|---|---|
| 资本控制生产 | 工具、原料、订单和生产决策由资本所有者/经理控制 |
| 稳定工资依赖 | 大量劳动者主要靠工资而非自有土地/身份供养 |
| 产品市场化 | 生产主要为了区域、全国或海外市场销售 |
| 利润再投资 | 盈余持续进入设备、规模、技术和管理 |
| 集体制度化 | 企业、会馆、公所、合同法院、商会或政治代表保护关系 |

每项 0—100。三项超过中阈值且没有严重断环，可出现“资本主义关系扩张”；四至五项长期高值，才成为主导生产关系。它仍可能与地主、国家和身份关系混合。

## 11. 五种资本主义/工业化路径

### 11.1 士商融合

商人购地、教育子弟和进入官僚，士绅投资贸易与作坊。

- 稳定性高、政治合法性强；
- 资本容易回流土地，工业再投资不足；
- 地方士绅与商业法权可能形成寡头联盟。

### 11.2 商人—工业资本

商帮把信用和市场网络转入集中工场、航运和设备。

- 市场响应快、创新动力强；
- 需要合同法权和劳动力；
- 工资冲突、垄断和资本俘获国家风险高。

### 11.3 国家资本/发展国家

官府控制战略工业、交通、教育和长期投资，私人企业围绕国家订单成长。

- 能突破基础设施和军工瓶颈；
- 官僚效率、预算纪律和公共监督决定成败；
- 可能演化为军政工业集团。

### 11.4 军商承包资本

将领、军需商、矿冶和边贸在战争需求下结合。

- 能在外压下快速积累；
- 和平转型困难，依赖军费与特许；
- 容易形成地方割据、强制劳动和战争资本主义。

### 11.5 合作/公共工业

工匠联合、地方公共机构、村社和劳动组织共同拥有或管理生产。

- 分配较平、组织参与强；
- 融资、专业管理和跨区协调困难；
- 可以与国家或市场并存，不要求一次消灭所有私人资本。

## 12. 劳动制度选择

| 制度 | 生产成本 | 技能/忠诚 | 组织后果 | 风险 |
|---|---|---|---|---|
| 身份强制工役 | 现金低 | 技能流失、逃役 | 国家/中介控制 | 低效、反抗、无法扩展 |
| 短期雇工 | 灵活 | 培训意愿低 | 劳动分散 | 波动、质量和季节缺工 |
| 稳定工资工场 | 成本可见 | 技能积累 | 工人组织上升 | 工资、罢工和失业政治 |
| 军事化劳动 | 战时集中 | 纪律高、创新受限 | 军政控制 | 强制、伤亡和军头化 |
| 行会/作坊自治 | 质量与技能好 | 身份认同强 | 工匠组织 | 准入封闭、规模限制 |
| 合作管理 | 分配与参与高 | 忠诚高 | 劳动政治强 | 决策慢、资本不足 |

工业化不等于劳动者自动获益。工资、工时、救济、迁徙、组织权和政治参与分别决定劳动集团对路线的支持。

## 13. 工业化的危机机制

### 13.1 粮价—工资危机

城市和工场扩张快于农业、运输和仓储，粮价上升；雇主压工资则工人生计崩溃，提工资则利润和出口受损。解决方案包括农业生产、粮运、工资协商、救济和产业升级，而非单一补贴。

### 13.2 信用与债务危机

工程和企业依赖商人垫款，战争或贸易中断后违约；国家可救助、重组、通胀、没收或允许破产，每种选择重新分配损失。

### 13.3 过度军工化

军队订单支撑工业，但民用需求和财政不足；和平后工场停产、失业和军工集团阻挠裁军。转民用需要技术、市场和政治交易。

### 13.4 区域不平衡

资本、人才和交通集中在少数区域，内地失地人口外流，地方财政和政治不满上升。转移支付、基础设施和产业扩散会降低核心利润并触发反对。

### 13.5 环境与资源约束

矿冶、木材、燃料和水利扩张提高产能，也造成土地、水源、灾害和社区冲突。资源不是无限的地图加成。

### 13.6 劳动政治

共同劳动和传播提高组织度，工人从个别逃亡转向工资谈判、罢工和政治代表。镇压降低短期组织但可能提高共同记忆与激进化。

## 14. 科研、教育与意识形态

技术教育会制造新的技术官僚、工匠精英和劳动骨干。科举专业化可以扩大国家技术能力，却会触发经典科举网络反对；私人书院和商办学校促进扩散，也可能形成阶级封闭。

技术纲领会与不同意识形态结合：

- 皇统富强：技术服务皇权和国防；
- 官僚实学：专业治理、统计和公共工程；
- 商业实用：利润、航海、合同和市场；
- 民生实学：水利、农业、公共卫生和减轻劳动；
- 劳动解放：机器降低苦役、合作管理和公共福利；
- 军事救国：纪律、军工和国家生存优先。

同一技术不会自动决定政治方向。

## 16. AI 改革策略

AI 首先判断“解决什么危机”，再选路线，不按固定科技顺序：

```text
reform_score = crisis_relevance
             + coalition_support
             + expected_state_survival
             + leader_program
             + regional_fit
             - fiscal_gap
             - implementation_gap
             - empowered_rival_risk
```

AI 会优先做一至两个主改革，避免同时启动十个工程。紧急战争可打断民用项目，但留下停工、债务和失业后果。继任者不会无条件取消前任路线，而根据沉没成本、支持联盟和个人纲领选择继续、修订、冻结或清算。

## 17. 验收清单

- [ ] 每项改革案填写问题、目标、发起者、赢家、输家、资源、失败和维持条件；
- [ ] 改革有调查、联盟、法令、执行、审计和制度再生产阶段；
- [ ] 强制力不能替代财政、人员、信息和社会合作；
- [ ] 工业化需要七环闭合，单一创新或建筑不能完成转型；
- [ ] 五类国家能力从区域、建筑和项目聚合，不逐件模拟商品；
- [ ] 知识只降低试验成本，必须制度化才能跨人物继承；
- [ ] 每种融资方式改变未来权力关系；
- [ ] 资本可流向土地、贸易、工业或国家军需，私人资本不可被直接命令；
- [ ] 资本主义判定基于五项生产关系并允许混合；
- [ ] 五条资本/工业路径均有优势、矛盾和失败状态；
- [ ] 工资、组织、失业和民生独立于总产能；
- [ ] 工业化会产生粮价、信用、军工、区域、资源和劳动危机；
- [ ] AI 同时维持的主改革数量受限，并考虑沉没成本和继承。

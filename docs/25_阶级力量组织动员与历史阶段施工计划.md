# 25 阶级力量、组织动员与历史阶段施工计划

> 前置硬约束：[24 轻量数值博弈与弃用范围](24_轻量数值博弈与弃用范围.md)。本系统不模拟人口、财富、产业或地区阶级构成，只模拟全国政治力量怎样形成、结盟、分裂并改变制度。

## 1. 核心数据不再扩张

直接使用 `24` 冻结的八项全国变量，不再增设八套阶级值。阶级/集团是这些值的解释层：

| 政治主体 | 主要读取 | 主要行为 |
|---|---|---|
| 皇帝与内廷 | 原模组皇权、`elite_cohesion`、`state_capacity` | 仲裁、强推、分化外朝 |
| 官僚士绅 | `conservative_power`、`elite_cohesion` | 执行、拖延、朋党组织 |
| 军事集团 | `military_autonomy`、危机压力 | 提供强制力、索取自主权 |
| 城乡民众 | `popular_mobilization`、危机压力 | 请愿、抗拒、起事、革命动员 |
| 商业力量 | `commercial_influence`、改革势头 | 提供政治支持、要求制度保障 |
| 改革派网络 | `reform_momentum`、国家能力 | 把观念变为组织和政策 |

这些主体不是人物身份特质。人物层完全优先读取《变身大明》已有官职、品级、朋党、军职、锦衣卫、宦官等身份；没有必要时不新增 trait。

## 2. 年度变化规则

八项数值不做自动复杂演算。只有三类来源能改变它们：

1. 政策结算：一次修改 1–3 项，每项通常 `±2..8`；
2. 重大危机阶段：每阶段 `±3..10`；
3. 王位更替/战争结果等本体事实：一次性冲击。

季度脉冲只在活动政策或危机存在时运行。无活动内容时不自然增长；年度维护仅执行一次钳制、阶段判定和兼容探针。

统一修改接口：

```text
minghm_change_power_effect = {
  TYPE = 固定枚举
  DELTA = -20..20
  REASON = 本地化原因编号
}
```

effect 内完成 `0..100` 钳制，并记录最近一次原因供事件提示。禁止各事件直接散写变量。

## 3. 动员是一次性状态，不是人口池

一场政策/危机可选择一次动员方式：

| 方式 | 立即修正 | 风险 |
|---|---|---|
| 官僚动员 | 国家能力加入推动力 | 受守成力量与朋党阻挠 |
| 皇权动员 | 消耗原模组皇权，快速加推动力 | 精英协调下降、强推反噬 |
| 军事动员 | 军事能力解决短期问题 | 军事自主上升 |
| 民众动员 | 民众动员加入改革/革命推动力 | 结果更难由皇帝完全控制 |
| 商业协力 | 商业影响加入改革推动力 | 商业力量政治要求上升 |
| 厂卫威慑 | 压低当期公开阻力 | 信息失真、恐惧与后续消极执行 |

动员方式保存在活动槽 `minghm_active_mobilization = 0..6`，事件结束清零。不存在逐人物成员、组织花名册或地区动员率。

## 4. 历史阶段只保留五档

`minghm_historical_stage = 0..4`：

| 阶段 | 组合条件（连续两个年度判定才改变） | 主要玩法 |
|---|---|---|
| 0 传统秩序稳定 | 守成强、精英协调高、危机低 | 小改革成本低，深改缺乏动力 |
| 1 秩序失衡 | 危机或军事自主升高 | 皇帝在修补、压制、改革间选择 |
| 2 改革竞争 | 改革势头与守成力量都较高 | 长期非一边倒政策博弈 |
| 3 双重权力 | 民众动员/军事自主极高且精英分裂 | 朝廷外行动者能左右结果 |
| 4 新秩序整合 | 某路线胜出且国家能力恢复 | 处理承诺、反弹和制度化 |

阶段不是线性科技树，可前进、停滞和倒退。阶段只控制事件池、AI 权重和政策门槛，不直接大幅加成属性。

### 判定示例

```text
改革竞争分 = reform_momentum + popular_mobilization/2
旧秩序分   = conservative_power + elite_cohesion/2
解体风险分 = crisis_pressure + military_autonomy - state_capacity
```

比较分差与门槛即可，不用矩阵或逐阶级积分。年度第一次满足只置 `minghm_stage_candidate`；下一年度仍满足才正式切换，防止来回抖动。

## 5. 非一边倒反馈

每项强力策略必须同时产生一种长期代价：

- 加强国家能力可能加强掌握国家机器的既有官僚；
- 扩大民众动员会增强改革，却也使皇帝不能随意终止进程；
- 借军队破局会提高军事自主；
- 借商人支持制度创新会提高商业政治影响；
- 依赖皇权绕过协商会削弱精英协调；
- 与守成力量妥协能恢复秩序，但降低改革势头或改革深度。

这套对价是主要“数字游戏”，不追加复杂资源币。

## 6. 脚本落位

```text
common/scripted_effects/minghm_power_effects.txt
common/scripted_values/minghm_power_values.txt
common/scripted_triggers/minghm_stage_triggers.txt
common/on_action/minghm_stage_on_actions.txt
events/minghm_stage_events.txt
localization/simp_chinese/minghm_stage_l_simp_chinese.yml
```

年度 `on_action` 只从 `title:h_greatming` 进入一次。所有 trigger 首先调用 `minghm_is_active_ming_system_trigger`。

## 7. 验收

- 100 年观察中八项变量均保持 `0..100`，阶段保持 `0..4`；
- 不存在地区、商品、人口、逐人物阶级变量；
- 阶段能倒退，且不会因一年波动连续跳档；
- 每条动员路线至少有一个对抗性长期代价；
- 非明统治者从不运行阶段判定或获得相关事件。

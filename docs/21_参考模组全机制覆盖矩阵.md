## 4. 明确排除

即使原模组或 DLC 已有相似功能，本项目也不实现或扩展：财政/国库预算、地区生产、土地矿业、商品市场、人口迁徙、跨政权朝贡殖民、八旗国家玩法、新兵种军器库存、第二套政体官阶，以及面向其他国家的通用框架。

## 5. 反查流程

1. 从 `27A` 确认该机制在施工队列；
2. 在本表选择事实资料；
3. 用 `02/02A` 定位精确键与调用链；
4. 用对应施工书和 `27` 核对本体样板、覆盖和美术边界；
5. 回到对应 `23—30` 施工书实现；
6. 探针未通过时执行施工书规定的回退，不从已排除机制中另造系统。

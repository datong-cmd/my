# 30A 事件链一：新君临朝与三司定局

## 1. 入口与目标

入口：`on_title_gain` 中新 holder 获得 `h_greatming`，且不是同一人物重复授予；若已有其他主链，继承事件优先接管并把旧链转为本链的“遗留议题”。

目标：让继承不只是威望/皇权增减，而是决定新君依靠外朝官僚、内廷、军政力量还是多方制衡。全链不改政府类型，不要求天朝制。

初值：

```text
support = 60 + elite_cohesion/2 + 新君与关键官员关系修正
resistance = 40 + crisis_pressure/2 + conservative_power/3
urgency = 70（每阶段下降，拖延事件可回升）
commitment = 0
```

## 3. 路线结果

| 路线 | 短期 | 长期对价 |
|---|---|---|
| 外朝共治 | 支持和国家能力上升 | 守成力量获得否决入口，皇帝强推空间变小 |
| 内廷主导 | 诏令速度快、公开阻力低 | 精英协调下降、信息失真风险上升 |
| 军政担保 | 危机响应上升 | 军事自主上升 |
| 皇帝亲裁 | 消耗皇权换支持 | 后续失败由皇帝独担，党争地下化 |
| 多方制衡 | 避免单方坐大 | 政策推动力较低，下一链更易触发党争 |

## 4. 角色和接口

优先抓取大学士/首辅、左都御史、司礼监掌印、五军职位代表；空缺就显示“官署空悬”。皇权消费只通过 `minghm_spend_hq_effect`；不改原职位、不调用原任官事件。

主题：`crown` 开场，`zouzhang` 廷议，`guanwei` 名册。背景 `dadian/gongdian/shufang`。

## 5. 结算

`support - resistance ≥25` 为稳固定局；`5..24` 为妥协；`-15..4` 为权力悬空；`<-15` 触发摄政/党争余波。结算修改国家能力、精英协调、军事自主、守成力量各 1–3 项，总幅度不超过 12。

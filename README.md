# 明代权力运行与历史变迁 Mod 策划

本仓库保存 Crusader Kings III 明代政治、社会经济转型与改革革命 Mod 的分析和策划文档。

设计重点是长期、非一边倒的权力博弈，并以历史唯物主义组织生产力、生产关系、阶级力量、国家能力、意识形态和历史阶段的变化。史实用于提供矛盾结构与事件素材，玩法、可读性、AI 可运行性和性能优先于逐年复刻。

## 文档入口

从 [总目录与阅读路线](docs/00_总目录与阅读路线.md) 开始。

当前阶段已经包含：

- 参考模组全量源码文件、脚本 ID、事件调用和资产索引；
- 参考模组基础架构、触发入口与性能初判；
- 流官、科举、授官与考成机制；
- 皇权、内阁、票拟、封还与首辅摄政；
- 朋党组织、政治影响力与权臣形成；
- 新 Mod 的历史唯物主义方法、长期博弈原则和初步技术边界。

## 仓库边界

- `docs/`：可交付的 Markdown 分析与策划资料。
- 不上传 Crusader Kings III 本体文件、创意工坊模组源文件、PDF 原件或大型美术资产。
- 文档中的文件路径、脚本 ID 和行号用于合法本地安装中的实现检索，不代表仓库包含对应受版权保护的源文件。
- 自动生成的索引证明源码中存在相应定义或调用；运行时可达性仍需结合调用链和实机日志验证。

本仓库按阶段发布：每完成一个可独立复核的机制阶段，更新文档、通过链接与编码检查后提交到 `main`。

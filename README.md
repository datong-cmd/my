# 明代权力运行：变身大明伴生扩展策划

本仓库是 CK3 `1.19.0.6` + All Under Heaven +《变身大明》（Workshop `2713348525`）的明制专用兼容扩展计划书。

项目只处理 `h_greatming` 内部政治：皇权与官僚执行、朋党与监察、阶级力量、历史阶段、改革、技术组织、重大危机和革命/反革命。玩法采用少量全国数值和长期事件博弈，性能与可解释性优先。

明确不做：财政和国库预算、区域生产、市场商品、人口模拟、其他国家通用化、新政体/兵种/建筑、全地图扫描、大型公共 GUI 覆盖或 3D 复制。

## 入口

- 人工阅读：[总目录与阅读路线](docs/00_总目录与阅读路线.md)
- 后续 AI 编码：[27A AI 编码交付总实施书](docs/27A_AI编码交付总实施书.md)

`docs/01—10` 中保留的是源码、调用链、机制和资产事实，用于按需查证；它们不自动成为实现需求。真正的施工范围以 `22—30G` 为准。

仓库不包含 CK3 本体、DLC、创意工坊 Mod 源文件或二进制美术资产。
